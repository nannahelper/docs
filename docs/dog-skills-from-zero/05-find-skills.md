# 第 5 章：Find-Skills 技能发现

> **探索无限可能** —— 用 Find-Skills 发现和安装社区技能，持续扩展 AI 助手的能力边界。

---

## 5.1 Find-Skills 简介

Find-Skills 是 Dog-Skills 中的技能发现系统。当你需要完成某个任务但不确定 AI 助手是否具备相关能力时，Find-Skills 可以帮你搜索、推荐和安装合适的技能。

### 核心功能

| 功能 | 说明 |
|:---|:---|
| **技能搜索** | 根据任务描述搜索匹配的技能 |
| **技能推荐** | 基于需求推荐最合适的技能 |
| **技能安装** | 引导完成技能的安装流程 |
| **技能发现** | 浏览可用的技能生态 |

---

## 5.2 触发方式

Find-Skills 的触发非常自然，只需像平常一样描述你的需求即可：

### 通用触发词

- "如何做 X？"
- "我想找一个能帮我做 X 的技能"
- "有没有技能可以处理 X？"
- "how do I do X？"
- "is there a skill that can..."

### 示例对话

```
用户: "我想找一个能帮我处理 PDF 文档的技能"
AI:  [加载 Find-Skills 技能]
     我帮你搜索了可用的技能，找到了以下相关技能：
     1. pdf — PDF 文档处理技能
     2. ...
```

## 5.3 技能搜索流程

```mermaid
flowchart LR
```

## 5.4 实战：发现新技能

### 场景一：搜索 PDF 处理技能

```
用户: "我有一批 PDF 需要提取文字，有技能可以帮我吗？"
AI:  "我找到了 'pdf' 技能，它可以帮你：
     - 提取 PDF 中的文字内容
     - 合并、拆分 PDF 文件
     - 转换 PDF 为其他格式
     需要我帮你安装这个技能吗？"
```

### 场景二：搜索 Excel 处理技能

```
用户: "有没有能处理 Excel 表格的技能？"
AI:  "找到了 'xlsx' 技能，它可以帮你：
     - 读取和写入 Excel 文件
     - 数据分析与可视化
     - 公式计算与图表生成
     需要我帮你安装吗？"
```

### 场景三：无匹配技能

```
用户: "我想找一个能帮我做视频剪辑的技能"
AI:  "目前没有找到完全匹配的技能。你可以：
     1. 使用 Skill-Creator 创建一个自定义技能
     2. 在社区中提交技能需求
     3. 等待社区贡献相关技能"
```

## 5.5 技能安装流程

Find-Skills 找到技能后，会引导你完成安装：

### 步骤 1：确认技能信息

AI 会展示技能的详细信息：
- 技能名称和描述
- 功能列表
- 使用方法
- 依赖要求

### 步骤 2：下载技能

技能通常从 GitHub 仓库或其他源获取：

```bash
# 示例：安装 pdf 技能
git clone https://github.com/example/pdf-skill.git
cp -r pdf-skill ~/.agents/skills/pdf
```

### 步骤 3：验证安装

用技能的关键词触发，确认安装成功。
