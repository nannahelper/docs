# AI 编程助手技能系统：从发现到创作

> **让 AI 助手真正懂你** —— 掌握 AI 编程助手的技能（Skills）系统，从使用者进阶为创作者。

---

## 教程简介

现代 AI 编程助手（如 Claude Code、Cursor、Trae IDE 等）都支持**技能（Skills）系统**——通过加载结构化的领域知识和工作流程，让 AI 助手在特定任务上表现出专家级水平。

本教程将带你从零开始，掌握技能的**使用**、**发现**与**创作**三大核心能力，让 AI 助手真正成为你的超级生产力工具。

!!! info "为什么要学技能系统？"

    - **提升 AI 效率**：让 AI 助手按结构化流程工作，输出更专业、更可控
    - **复用最佳实践**：将优秀的工作流程固化为技能，一次创建，反复使用
    - **持续扩展能力**：通过技能生态不断为 AI 助手解锁新能力
    - **贡献社区**：将你的领域知识封装为技能，帮助更多人

| 项目 | 内容 |
|:---|:---|
| **学习目标** | 掌握 AI 技能系统的使用、发现与创作，能独立利用技能提升开发效率 |
| **预计时长** | 3~4 小时 |
| **前置知识** | 基本的 AI 编程助手使用经验（如 Claude Code、Cursor、Trae IDE 等） |
| **最终产出** | 能熟练使用技能系统，并创建自己的自定义技能 |

## 主流技能平台概览

| 平台 | 技能机制 | 特点 |
|:---|:---|:---|
| **Claude Code** | Skills（内置 + 自定义） | Anthropic 官方 CLI，生态最丰富 |
| **Cursor** | Rules + Skills | 灵活的规则系统，可模拟技能机制 |
| **Trae IDE** | Skills 目录 | 原生 Skills 支持，拖拽即用 |
| **Codex** | Skills + 分发协议 | 可与 Claude Code 协同的调度系统 |
