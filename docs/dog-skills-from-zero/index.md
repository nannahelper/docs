# Dog-Skills 智能技能系统使用指南

> **让 AI 技能成为你的超级助手** —— 从零掌握 Dog-Skills 智能技能系统，释放 AI 编程助手的全部潜力。

---

## 教程简介

Dog-Skills 是一套面向 AI 编程助手（如 Trae IDE、Cursor 等）的智能技能系统，通过模块化的技能包，为 AI 助手注入专业领域知识和工作流程。无论是生成高质量教程、设计前端界面，还是发现和安装新技能，Dog-Skills 都能让 AI 助手从"通用助手"升级为"领域专家"。

!!! info "为什么学 Dog-Skills？"

    - **提升 AI 效率**：让 AI 助手按结构化流程工作，输出更专业、更可控
    - **开箱即用**：每个技能都是独立模块，安装即用，无需复杂配置
    - **持续扩展**：技能生态持续增长，覆盖教程生成、前端设计、技能发现等场景
    - **社区驱动**：开源项目，欢迎贡献新技能和改进现有技能

| 项目 | 内容 |
|:---|:---|
| **学习目标** | 掌握 Dog-Skills 的安装与使用方法，能独立利用技能系统提升开发效率 |
| **预计时长** | 3~4 小时 |
| **前置知识** | 基本的 AI 编程助手使用经验（如 Trae IDE、Cursor 等） |
| **最终产出** | 能熟练使用 Dog-Skills 三大核心技能完成实际任务 |

## 技能概览

| 技能 | 图标 | 功能 | 典型场景 |
|:---|:---:|:---|:---|
| **dog-tutor** | 📚 | 智能教程生成 | 生成技术文档、课程大纲、入门指南 |
| **dog-frontier** | 🎨 | 前端设计综合 | UI/UX 设计、Tailwind/shadcn 组件、落地页 |

## 快速开始

```bash
# 1. 克隆 Dog-Skills 仓库
git clone https://github.com/ZhouYinLong-lab/Dog-Skills.git

# 2. 进入技能目录
cd Dog-Skills

# 3. 将技能复制到 AI 助手的技能目录（以 Trae IDE 为例）
# 技能目录通常在 ~/.agents/skills/ 或类似路径
```
