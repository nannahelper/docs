## 教程简介

MATLAB（Matrix Laboratory）是由 MathWorks 公司开发的数值计算与编程平台，自 1984 年发布以来已成为工程和科学领域的标准工具。其核心优势在于将矩阵运算、数据可视化和算法开发集成在一个易用的环境中。

!!! info "为什么学 MATLAB？"

    - **矩阵运算原生支持**：MATLAB 的名字就是"矩阵实验室"，矩阵操作是其灵魂
    - **丰富的工具箱**：信号处理、图像处理、控制系统、机器学习等专业工具箱
    - **Simulink 仿真**：图形化建模与仿真环境，工程领域的事实标准
    - **学术界广泛使用**：许多高校的工科课程和科研项目以 MATLAB 为平台

## 快速预览

```matlab
% 生成数据
x = linspace(0, 2*pi, 100);    % 0 到 2π 的 100 个等分点
y1 = sin(x);
y2 = cos(x);

% 绘图
figure;
plot(x, y1, 'b-', 'LineWidth', 2);   % 蓝色实线
hold on;                               % 保持图形
plot(x, y2, 'r--', 'LineWidth', 2);   % 红色虚线
hold off;

% 图形美化
xlabel('x (弧度)');
ylabel('函数值');
title('正弦与余弦函数');
legend('sin(x)', 'cos(x)');
grid on;
```

**渲染效果：** 生成一个图形窗口，显示蓝色实线的正弦曲线和红色虚线的余弦曲线，包含坐标轴标签、标题、图例和网格线。
