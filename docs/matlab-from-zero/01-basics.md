## 1.1 MATLAB 的定位

MATLAB 是一个**面向矩阵运算**的数值计算平台。与 Python 和 R 不同，MATLAB 从底层设计就以矩阵为核心数据结构。

| 对比维度 | MATLAB | Python + NumPy | R 语言 |
|:---|:---|:---|:---|
| **核心定位** | 数值计算与工程仿真 | 通用编程 + 科学计算 | 统计计算 |
| **矩阵运算** | 原生语法，最自然 | NumPy 提供，需导入 | 需 matrix 类型 |
| **可视化** | 内置强大，交互式 | matplotlib 等库 | ggplot2 |
| **工具箱** | 商业授权，专业完善 | 开源生态 | CRAN 包 |
| **Simulink** | 独有，图形化仿真 | 无等价替代 | 无 |
| **费用** | 商业软件（有学生版） | 免费开源 | 免费开源 |

!!! tip "MATLAB 适合谁？"

    如果你是工科学生（电子、机械、控制、通信等），MATLAB 几乎是必修课。如果你主要做通用数据科学，Python 或 R 可能更合适。

## 1.2 MATLAB 工作界面

MATLAB 桌面环境包含以下主要组件：

| 组件 | 位置 | 功能 |
|:---|:---|:---|
| **命令窗口（Command Window）** | 中央 | 交互式执行命令，显示输出 |
| **编辑器（Editor）** | 中央（标签页） | 编写和调试脚本（`.m` 文件） |
| **工作区（Workspace）** | 右侧 | 查看和管理变量 |
| **当前文件夹（Current Folder）** | 左侧 | 浏览和管理文件 |
| **路径（Path）** | 工具栏 | 管理 MATLAB 搜索路径 |

!!! tip "常用快捷键"

    - `F5`：运行当前脚本
    - `F9`：运行选中的代码
    - `Ctrl + C`：中断正在执行的命令
    - `clc`：清空命令窗口（不删除变量）
    - `clear`：清空工作区变量

## 1.3 第一个 MATLAB 程序

```matlab
% 这是我的第一个 MATLAB 程序
disp('Hello, MATLAB!');

% 基本运算
2 + 3           % 加法 → 5
10 / 3          % 除法 → 3.3333
2^10            % 幂运算 → 1024
sqrt(16)        % 平方根 → 4

% 变量赋值
x = 100;
y = 200;
name = '张三';

% 查看变量
disp(x);
fprintf('你好，%s！\n', name);   % 格式化输出

% 查看变量信息
whos;           % 列出所有变量及其类型和大小
```

**运行效果：** 命令窗口依次显示每行的输出。`%` 是注释符号，`;` 分号抑制输出（不加分号会显示结果）。

## 1.4 帮助系统

MATLAB 拥有完善的帮助文档：

```matlab
help sin         % 查看 sin 函数的帮助
doc sin          % 在帮助浏览器中打开详细文档
lookfor matrix   % 搜索与 "matrix" 相关的函数
```
