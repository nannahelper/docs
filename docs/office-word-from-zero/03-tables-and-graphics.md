# 第 3 章：表格与图形

> **从"纯文字"到"图文并茂"——让文档更具表现力和说服力**

一份优秀的文档不只是文字的堆砌。表格让数据清晰可读，图形让概念直观易懂。本章教你如何让 Word 文档"活"起来。

---

## 3.1 表格的高级应用

### 表格的三种创建方式

| 方式 | 操作 | 适用场景 |
|:---|:---|:---|
| **网格插入** | 插入 → 表格 → 拖动网格 | 快速创建规则表格 |
| **插入表格对话框** | 插入 → 表格 → 插入表格 | 需要指定行列数 |
| **绘制表格** | 插入 → 表格 → 绘制表格 | 创建不规则表格 |
| **文本转换为表格** | 选中文本 → 插入 → 表格 → 文本转换成表格 | 已有数据快速制表 |
| **Excel 电子表格** | 插入 → 表格 → Excel 电子表格 | 需要计算功能的表格 |

### 表格样式与设计

!!! tip "表格设计原则"
    - **标题行加粗** ：让表头与数据行形成视觉层次
    - **隔行变色** ：使用"镶边行"提高长表格的可读性
    - **对齐规则** ：文字左对齐，数字右对齐，标题居中对齐
    - **去除冗余边框** ：使用简洁的边框设计，避免"铁笼"效果

!!! example "实战：制作专业数据表格"
    1. 插入表格后，选中表格
    2. 表格设计 → 选择"网格表 4 - 强调文字颜色 1"
    3. 勾选"标题行"、"镶边行"、"第一列"
    4. 手动调整列宽：双击列边界自动适配内容宽度
    5. 数字列设置右对齐，文字列保持左对齐

### 表格中的公式计算

Word 表格支持基本的公式计算，类似 Excel 的简化版。

| 常用函数 | 用途 | 示例 |
|:---|:---|:---|
| `SUM` | 求和 | `=SUM(ABOVE)` 对上方单元格求和 |
| `AVERAGE` | 平均值 | `=AVERAGE(LEFT)` 对左侧单元格求平均 |
| `MAX` / `MIN` | 最大/最小值 | `=MAX(ABOVE)` |
| `COUNT` | 计数 | `=COUNT(LEFT)` |

!!! info "单元格引用"
    Word 表格的列用字母（A、B、C...），行用数字（1、2、3...）。`A1` 表示第一列第一行，`B2:C4` 表示 B2 到 C4 的矩形区域。

## 3.2 SmartArt：概念可视化

### SmartArt 的八种类型

| 类型 | 用途 | 典型场景 |
|:---|:---|:---|
| **列表** | 展示非顺序信息 | 要点罗列、目录 |
| **流程** | 展示步骤或时间线 | 工作流程、操作步骤 |
| **循环** | 展示循环往复的过程 | PDCA 循环、生命周期 |
| **层次结构** | 展示层级关系 | 组织架构图、分类树 |
| **关系** | 展示元素之间的关联 | 韦恩图、对比图 |
| **矩阵** | 展示四象限关系 | SWOT 分析、优先级矩阵 |
| **棱锥图** | 展示比例或层级关系 | 需求层次、漏斗分析 |
| **图片** | 图文结合的展示 | 团队介绍、产品展示 |

!!! example "实战：用 SmartArt 制作组织架构图"
    1. 插入 → SmartArt → 层次结构 → 组织结构图
    2. 输入各层级名称
    3. 使用"设计"选项卡切换配色方案
    4. 右键形状 → 添加形状 → 在下方添加（增加下属）
    5. 使用"格式"选项卡调整形状样式

## 3.3 图片与图文混排

### 图片的六种文字环绕方式

| 环绕方式 | 行为 | 适用场景 |
|:---|:---|:---|
| **嵌入型** | 图片像一个大字符，随文字移动 | 简单插图 |
| **四周型** | 文字环绕图片矩形边界 | 常规图文混排 |
| **紧密型** | 文字紧贴图片轮廓 | 去背景的图片 |
| **穿越型** | 文字填充图片透明区域 | 特殊效果 |
| **上下型** | 图片独占一行，文字在上下 | 大图展示 |
| **衬于文字下方** | 图片作为背景 | 水印、背景图 |
| **浮于文字上方** | 图片覆盖文字 | 标注、角标 |

!!! tip "图片压缩"
    插入 → 图片 → 选择图片后，在"图片格式"选项卡中点击"压缩图片"。选择"电子邮件（96 ppi）"可大幅减小文档体积，适合电子分发。

### 图片样式与效果

| 效果 | 说明 | 快捷键/路径 |
|:---|:---|:---|
| **图片样式库** | 一键应用预设边框和效果 | 图片格式 → 图片样式 |
| **图片边框** | 自定义边框颜色和粗细 | 图片格式 → 图片边框 |
| **图片效果** | 阴影、反射、发光、柔化边缘 | 图片格式 → 图片效果 |
| **裁剪** | 裁剪为形状或按比例裁剪 | 图片格式 → 裁剪 |

## 3.4 图表：数据可视化

### 在 Word 中插入图表

Word 中的图表本质上是嵌入了一个简化的 Excel 工作表。

!!! example "实战：插入柱形图展示销售数据"
    1. 插入 → 图表 → 柱形图 → 确定
    2. 在弹出的 Excel 窗口中修改数据
    3. 关闭 Excel 窗口，图表自动更新
    4. 使用"图表设计"选项卡切换样式
    5. 使用"图表格式"选项卡微调元素

### 图表元素

| 元素 | 说明 | 建议 |
|:---|:---|:---|
| **图表标题** | 描述图表内容 | 简洁明了，不超过 15 字 |
| **数据标签** | 直接显示数值 | 数据点少时使用 |
| **图例** | 说明数据系列 | 放在图表下方或右侧 |
| **坐标轴标题** | 说明坐标轴含义 | 有单位的务必标注 |
