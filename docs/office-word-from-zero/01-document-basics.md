# 第 1 章：文档基础操作与排版

> **从"打字员"到"排版师"——掌握 Word 的核心排版引擎**

很多人在 Word 里用空格对齐文字、用回车换页。本章将帮你摆脱这些"野路子"，建立正确的排版思维。

---

## 1.1 段落格式的深层控制

### 段落的三个核心属性

把每个段落想象成一个"盒子"，这个盒子有三个关键属性：

| 属性 | 作用 | 常见误区 |
|:---|:---|:---|
| **对齐方式** | 控制文字在水平方向的位置 | 用空格实现居中对齐 |
| **缩进** | 控制段落与页边距的距离 | 用 Tab 或空格手动缩进 |
| **间距** | 控制段落内部和段落之间的距离 | 用空行（连续回车）制造间距 |

!!! info "段落标记 ¶ 的重要性"
    按 `Ctrl+Shift+8` 显示/隐藏编辑标记。段落标记 ¶ 存储了该段落的 **所有格式信息** 。删除一个 ¶，就删除了该段落的全部格式。

### 缩进的四种类型

| 缩进类型 | 说明 | 典型用途 |
|:---|:---|:---|
| **首行缩进** | 仅段落第一行缩进 | 中文正式文档（通常 2 字符） |
| **悬挂缩进** | 除第一行外其余行缩进 | 参考文献列表、项目编号 |
| **左缩进** | 整个段落从左侧缩进 | 引用块、强调段落 |
| **右缩进** | 整个段落从右侧缩进 | 特殊排版需求 |

!!! example "实战：设置正文首行缩进 2 字符"
    1. 选中正文段落
    2. 右键 → 段落 → 缩进和间距
    3. 特殊格式 → 首行缩进 → 2 字符
    4. 点击确定

    此后所有新段落自动继承此格式，无需手动按空格。

### 段落间距 vs 行距

- **段前/段后间距** ：段落与相邻段落之间的距离（像楼层之间的高度）
- **行距** ：同一段落内行与行之间的距离（像同一楼层内的天花板高度）

!!! tip "推荐设置"
    中文正式文档推荐：首行缩进 2 字符 + 1.5 倍行距 + 段后 6 磅。这样既美观又不需要手动空行。

## 1.2 制表位：告别空格对齐

### 为什么不能用空格对齐？

空格是"比例字体"——不同字符宽度不同，空格无法保证精确对齐。制表位才是 Word 中的"隐形列线"。

### 制表位的四种类型

| 类型 | 图标 | 对齐方式 | 典型用途 |
|:---|:---:|:---|:---|
| **左对齐** | └ | 文字从制表位向右延伸 | 普通文本对齐 |
| **居中对齐** | ┴ | 文字以制表位为中心 | 标题居中 |
| **右对齐** | ┘ | 文字从制表位向左延伸 | 数字、日期对齐 |
| **小数点对齐** | ┴ | 以小数点为中心对齐 | 财务报表数字 |

!!! example "实战：制作目录式对齐的价目表"
    1. 在标尺上点击设置制表位（或双击标尺精确设置）
    2. 输入"项目名称"，按 `Tab`，输入"单价"
    3. 下一行输入"笔记本电脑"，按 `Tab`，输入"5999.00"
    4. 为价格列设置 **小数点对齐** 制表位

    效果：所有价格的小数点完美对齐，无论整数部分有几位。

## 1.3 分节符：文档的"隐形墙壁"

### 节的概念

**节** 是 Word 文档格式的基本管理单元。同一节内的页面共享相同的页面设置（纸张方向、页边距、页眉页脚等）。

!!! info "分节符 vs 分页符"
    - **分页符** （`Ctrl+Enter`）：仅换页，新页继承上一页的所有格式
    - **分节符** ：创建新的格式管理单元，新节可以有不同的页面设置

### 分节符的四种类型

| 类型 | 行为 | 典型用途 |
|:---|:---|:---|
| **下一页** | 新节从新页开始 | 每章从新页开始 |
| **连续** | 新节从当前页继续 | 同一页内切换分栏 |
| **偶数页** | 新节从下一个偶数页开始 | 书籍排版（章节从右页开始） |
| **奇数页** | 新节从下一个奇数页开始 | 书籍排版（章节从右页开始） |

!!! example "实战：论文中插入横向表格页"
    1. 在表格前插入"下一页"分节符
    2. 在表格后插入"下一页"分节符
    3. 将表格所在节的纸张方向设为横向
    4. 前后章节保持纵向

    这样只有表格页是横向的，其他页面不受影响。

## 1.4 页面设置与版心设计

### 版心概念

**版心** 是页面上实际放置内容的区域，由页边距围成。合理的版心设计直接影响阅读体验。

| 文档类型 | 推荐页边距 | 说明 |
|:---|:---|:---|
| 学术论文 | 左 3cm，右 2.5cm，上 2.5cm，下 2cm | 左侧留装订空间 |
| 商业报告 | 上下左右各 2.5cm | 均衡对称 |
| 手册/书籍 | 使用"对称页边距" | 内侧 > 外侧，考虑装订 |

!!! tip "对称页边距"
    在"页面设置 → 页边距 → 多页"中选择"对称页边距"，内侧边距自动大于外侧，适合双面打印和装订。
