## 7.1 域代码入门

### 什么是域？

**域** 是 Word 中的动态内容占位符，它根据当前状态自动生成内容。页码、目录、交叉引用——这些本质上都是域。

| 常用域 | 功能 | 示例 |
|:---|:---|:---|
| `PAGE` | 当前页码 | 页脚中的页码 |
| `NUMPAGES` | 文档总页数 | "第 X 页 / 共 Y 页" |
| `DATE` | 当前日期 | 自动更新的日期 |
| `TIME` | 当前时间 | 自动更新的时间 |
| `FILENAME` | 当前文件名 | 页眉中的文件名 |
| `DOCPROPERTY` | 文档属性 | 标题、作者等 |
| `IF` | 条件判断 | 邮件合并中的条件 |
| `SEQ` | 自动编号 | 自定义序列编号 |

!!! info "查看和编辑域代码"
    - `Alt+F9`：切换所有域的显示（域代码 ↔ 域结果）
    - `Shift+F9`：切换选中域的显示
    - `F9`：更新选中域
    - `Ctrl+A` → `F9`：更新文档中所有域
    - `Ctrl+F11`：锁定域（禁止更新）
    - `Ctrl+Shift+F11`：解锁域

### 常用域代码示例

!!! example "实战：页脚显示"第 X 页 / 共 Y 页""
    1. 双击页脚进入编辑
    2. 输入"第 "，按 `Ctrl+F9` 插入域括号
    3. 在括号中输入 `PAGE`
    4. 输入" 页 / 共 "
    5. 按 `Ctrl+F9`，输入 `NUMPAGES`
    6. 输入" 页"
    7. 按 `Alt+F9` 切换显示效果

!!! example "实战：自动显示文件名"
    在页眉中按 `Ctrl+F9`，输入：
    ```
    FILENAME \p
    ```
    `\p` 开关表示包含完整路径。去掉 `\p` 则只显示文件名。

## 7.2 通配符查找替换

### 启用通配符

查找和替换（`Ctrl+H`）→ 更多 → 勾选"使用通配符"。

### 常用通配符

| 通配符 | 含义 | 示例 | 匹配结果 |
|:---|:---|:---|:---|
| `?` | 任意单个字符 | `第?章` | 第1章、第A章 |
| `*` | 任意多个字符 | `第*章` | 第一章、第12章 |
| `[abc]` | 指定字符之一 | `[一二三]班` | 一班、二班、三班 |
| `[a-z]` | 范围内字符 | `[A-Z][0-9]` | A1、B2、Z9 |
| `{n}` | 重复 n 次 | `[0-9]{3}` | 123、456 |
| `{n,}` | 至少重复 n 次 | `[0-9]{2,}` | 12、12345 |
| `{n,m}` | 重复 n 到 m 次 | `[0-9]{2,4}` | 12、123、1234 |
| `@` | 重复 1 次或更多 | `[0-9]@` | 1、12、123 |
| `<` | 单词开头 | `<inter` | interest、internet |
| `>` | 单词结尾 | `ing>` | doing、going |
| `()` | 分组（用于替换） | `([0-9]{3})([0-9]{4})` | 分组捕获 |
| `\1` | 引用第 1 组 | 替换中使用 | 重新排列内容 |

!!! example "实战：批量删除空行"
    1. `Ctrl+H` 打开替换
    2. 勾选"使用通配符"
    3. 查找：`^13{2,}`
    4. 替换为：`^p`
    5. 全部替换

    这会将连续多个空行替换为单个空行。

!!! example "实战：电话号码格式化"
    将 `13812345678` 格式化为 `138-1234-5678`：
    
    查找：`([0-9]{3})([0-9]{4})([0-9]{4})`
    替换为：`\1-\2-\3`

## 7.3 常见故障排除

### 格式问题

| 问题 | 原因 | 解决方案 |
|:---|:---|:---|
| 页码不连续 | 分节符导致页码重置 | 页码 → 设置页码格式 → 续前节 |
| 目录显示错误 | 标题样式未正确应用 | 检查标题是否使用了内置标题样式 |
| 样式莫名变化 | 自动更新选项被勾选 | 修改样式 → 取消"自动更新" |
| 图片只显示一部分 | 行距设为"固定值" | 将行距改为"单倍行距"或"多倍行距" |
| 页眉横线去不掉 | 页眉段落有边框 | 选中页眉 → 开始 → 边框 → 无边框 |
| 文档体积过大 | 图片未压缩 | 压缩所有图片至 150 ppi |

### 性能问题

| 问题 | 解决方案 |
|:---|:---|
| 文档打开缓慢 | 检查是否有大量嵌入式对象；关闭自动拼写检查 |
| 滚动卡顿 | 切换到"草稿"视图（`Alt+Ctrl+N`） |
| 复制粘贴卡顿 | 关闭"显示粘贴选项按钮"（文件 → 选项 → 高级） |

### 崩溃恢复

!!! tip "找回未保存的文档"
    1. 文件 → 信息 → 管理文档 → 恢复未保存的文档
    2. 或直接访问：`%LOCALAPPDATA%\Microsoft\Office\UnsavedFiles`
    3. 设置自动保存间隔：文件 → 选项 → 保存 → 自动恢复间隔设为 5 分钟
