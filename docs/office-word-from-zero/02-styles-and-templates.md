# 第 2 章：样式与模板

> **从"逐段调格式"到"一键换装"——样式是 Word 排版的第一生产力**

如果你还在逐段设置字体、字号、颜色，那么你只发挥了 Word 10% 的排版能力。样式系统才是 Word 的核心引擎。

---

## 2.1 理解样式系统

### 什么是样式？

**样式** 是一组预定义的格式集合，像一个"格式配方"。应用一个样式，等于同时设置了字体、字号、颜色、段落间距、缩进等所有属性。

!!! info "样式的本质"
    把样式想象成 **CSS 中的 class** 。修改样式定义，所有应用该样式的内容自动更新——这就是"一键换装"的原理。

### 样式的层级结构

| 层级 | 样式类型 | 说明 | 示例 |
|:---|:---|:---|:---|
| 1 | **段落样式** | 控制整个段落的格式 | 正文、标题 1、标题 2 |
| 2 | **字符样式** | 仅控制选中文字的格式 | 强调、超链接、代码 |
| 3 | **链接样式** | 可同时用于段落和字符 | 列表段落 |
| 4 | **表格样式** | 控制表格的外观 | 网格表、清单表 |
| 5 | **列表样式** | 控制多级列表的格式 | 编号列表 |

### 样式继承关系

```
正文样式（基准样式）
  ├── 标题 1（基于正文 + 特定格式）
  ├── 标题 2（基于正文 + 特定格式）
  └── 自定义样式（基于正文 + 自定义格式）
```

!!! tip "基准样式的威力"
    修改"正文"样式的字体，所有基于正文的样式（标题 1、标题 2 等）的字体 **自动跟随变化** 。这就是为什么修改正文样式能影响整个文档。

## 2.2 创建与管理样式

### 创建新样式

!!! example "实战：创建"要点提示"样式"
    1. 打开样式窗格（`Ctrl+Alt+Shift+S`）
    2. 点击"新建样式"按钮
    3. 名称：要点提示
    4. 样式类型：段落
    5. 样式基于：正文
    6. 格式设置：字体微软雅黑、字号 11、左侧缩进 1cm、左侧边框（蓝色）
    7. 勾选"添加到样式库"和"基于该模板的新文档"

    此后在任何新文档中都能使用这个样式。

### 修改样式的三种方式

| 方式 | 操作 | 适用场景 |
|:---|:---|:---|
| **右键修改** | 在样式窗格中右键 → 修改 | 精确调整样式定义 |
| **更新以匹配所选内容** | 手动修改格式后，右键样式 → 更新 | 快速微调 |
| **格式刷** | 选中源格式 → 点击格式刷 → 刷目标 | 临时性格式复制 |

!!! warning "格式刷 vs 样式"
    格式刷是"一次性"的——复制后源和目标之间没有关联。样式是"持续性"的——修改样式定义，所有应用该样式的地方同步更新。 **能用样式解决的，不要用格式刷。**

## 2.3 样式集与主题

### 样式集

**样式集** 是一组样式的组合，可以一键切换整个文档的"视觉风格"。

| 操作 | 路径 |
|:---|:---|
| 切换样式集 | 设计 → 文档格式 → 选择样式集 |
| 保存自定义样式集 | 设计 → 文档格式 → 另存为新样式集 |
| 设为默认 | 设计 → 设为默认值 |

!!! example "实战：为不同场景准备样式集"
    创建三套样式集：
    - **正式报告** ：宋体正文 + 黑体标题 + 1.5 倍行距
    - **现代简约** ：微软雅黑正文 + 浅色标题 + 1.15 倍行距
    - **学术论文** ：宋体正文 + 严格行距 + 规范页边距

    切换样式集，整个文档瞬间"换装"。

### 主题三要素

| 要素 | 控制范围 | 说明 |
|:---|:---|:---|
| **颜色** | 文字颜色、形状填充、表格边框 | 一组 12 色的配色方案 |
| **字体** | 标题字体 + 正文字体 | 西文和中文分别设置 |
| **效果** | 形状、SmartArt 的视觉风格 | 阴影、反射、棱台等 |

## 2.4 模板：可复用的文档蓝图

### 模板 vs 普通文档

| 对比维度 | 普通文档（.docx） | 模板（.dotx） |
|:---|:---|:---|
| 打开方式 | 直接编辑 | 创建副本（保护原文件） |
| 样式和内容 | 一次性使用 | 每次新建都可用 |
| 存储位置 | 任意位置 | 自定义 Office 模板文件夹 |
| 适用场景 | 单次使用的文档 | 反复使用的文档类型 |

### 创建和使用模板

!!! example "实战：创建公司报告模板"
    1. 新建文档，设置好所有样式、页眉页脚、封面
    2. 在封面和正文中插入 **内容控件** （开发工具 → 控件）
    3. 文件 → 另存为 → 文件类型选择"Word 模板（*.dotx）"
    4. 保存到 `%USERPROFILE%\Documents\自定义 Office 模板`

    此后新建文档时，在"个人"模板中就能看到这个模板。

!!! tip "内容控件"
    在模板中插入内容控件（日期选择器、下拉列表、富文本等），使用者只需填写控件区域，不会误改模板结构。这就像表单中的"填空区"。
