# 第 8 章：综合实战项目

> **从零到交付——独立完成一份专业商业报告**

本章将前 7 章的所有技能整合到一个完整的实战项目中。你将扮演一家咨询公司的分析师，为"星辰科技"公司撰写一份年度市场分析报告。

---

## 8.1 项目需求分析

### 报告规格

| 要求 | 规格 |
|:---|:---|
| 文档类型 | 年度市场分析报告 |
| 目标读者 | 公司管理层和投资人 |
| 页数要求 | 15~20 页 |
| 核心内容 | 市场概况、竞争分析、财务数据、战略建议 |
| 交付格式 | .docx + PDF |

### 需要运用的技能

- 样式系统（统一全文格式）
- 分节符（封面、目录、正文分离）
- 自动目录（三级目录）
- 页眉页脚（奇偶页不同）
- 表格（财务数据展示）
- SmartArt（组织架构和流程图）
- 图表（市场趋势可视化）
- 交叉引用（引用图表编号）

---

## 8.2 第一步：搭建文档框架

### 创建样式体系

!!! example "操作步骤"
    1. 新建文档，打开样式窗格（`Ctrl+Alt+Shift+S`）
    2. 修改"正文"样式：宋体、小四、1.5 倍行距、首行缩进 2 字符
    3. 修改"标题 1"：黑体、二号、段前 12 磅、段后 6 磅
    4. 修改"标题 2"：黑体、三号、段前 6 磅、段后 6 磅
    5. 修改"标题 3"：黑体、小三、段前 6 磅、段后 6 磅
    6. 创建"表格正文"样式：宋体、五号、单倍行距、无缩进

### 设置分节结构

```
第 1 节：封面（无页眉页脚）
  ↓ 分节符（下一页）
第 2 节：目录（页码：I, II, III）
  ↓ 分节符（下一页）
第 3 节：正文（页码：1, 2, 3...）
  ├── 第 1 章 市场概况
  ├── 第 2 章 竞争分析
  ├── 第 3 章 财务分析
  ├── 第 4 章 战略建议
  └── 附录
```

## 8.3 第二步：撰写内容

### 封面设计

```
┌──────────────────────────────┐
│                              │
│     星辰科技                  │
│     2025 年度市场分析报告      │
│                              │
│     报告编号：XR-2025-001     │
│     编制日期：2025 年 12 月   │
│     密级：内部                │
│                              │
│     星辰咨询集团              │
└──────────────────────────────┘
```

!!! tip "封面设计技巧"
    - 使用"插入 → 形状"添加装饰线
    - 使用"页面颜色"设置背景色（打印时建议白色）
    - 封面不显示页眉页脚和页码

### 正文内容结构

| 章节 | 内容要点 | 使用的功能 |
|:---|:---|:---|
| 第 1 章 市场概况 | 行业规模、增长趋势、政策环境 | 图表（折线图、柱形图） |
| 第 2 章 竞争分析 | 主要竞争对手、市场份额、SWOT | SmartArt（矩阵）、表格 |
| 第 3 章 财务分析 | 收入、成本、利润、现金流 | 表格（公式计算）、图表 |
| 第 4 章 战略建议 | 短期策略、长期规划、风险提示 | SmartArt（流程）、编号列表 |

## 8.4 第三步：插入图表和数据

### 市场趋势图

!!! example "操作步骤"
    1. 插入 → 图表 → 折线图
    2. 在 Excel 窗口中输入数据：
    
    | 年份 | 市场规模（亿元） | 增长率 |
    |:---|:---|:---|
    | 2021 | 120 | — |
    | 2022 | 156 | 30% |
    | 2023 | 198 | 27% |
    | 2024 | 245 | 24% |
    | 2025 | 290 | 18% |
    
    3. 添加图表标题"行业市场规模与增长趋势"
    4. 右键图表 → 添加数据标签

### 财务数据表格

| 项目 | 2024 年 | 2025 年 | 同比变化 |
|:---|:---|:---|:---|
| 营业收入 | 8,500 | 10,200 | +20.0% |
| 营业成本 | 5,100 | 5,916 | +16.0% |
| 毛利润 | 3,400 | 4,284 | +26.0% |
| 净利润 | 1,700 | 2,228 | +31.1% |
| 毛利率 | 40.0% | 42.0% | +2.0pp |

!!! tip "表格美化"
    - 标题行使用深色背景 + 白色文字
    - 数据行使用"镶边行"（隔行变色）
    - 正增长用绿色，负增长用红色
    - 最后一列（同比变化）右对齐

## 8.5 第四步：添加目录和导航

### 生成目录

1. 将光标放在目录页
2. 引用 → 目录 → 自定义目录 → 显示级别 3
3. 生成后，右键目录 → 更新域 → 更新整个目录

### 添加图表目录

1. 为所有图表添加题注（右键 → 插入题注）
2. 引用 → 插入表目录 → 题注标签选择"图"
3. 同样方法创建"表目录"

---

## 8.6 第五步：页眉页脚设置

### 页眉方案

| 页面位置 | 页眉内容 |
|:---|:---|
| 奇数页（右页） | 当前章节标题 |
| 偶数页（左页） | "星辰科技 2025 年度市场分析报告" |

!!! example "实现方法"
    1. 双击页眉区域
    2. 页眉和页脚工具 → 勾选"奇偶页不同"
    3. 奇数页页眉：插入 → 文档部件 → 域 → StyleRef → 标题 1
    4. 偶数页页眉：手动输入报告标题

### 页码方案

- 封面：无页码
- 目录：罗马数字（I, II, III），页面底端居中
- 正文：阿拉伯数字（1, 2, 3），页面底端居中

---

## 8.7 第六步：最终审阅与导出

### 审阅清单

- [ ] 所有标题应用了正确的标题样式
- [ ] 目录页码与实际内容一致（更新目录）
- [ ] 所有图表有题注，正文中有交叉引用
- [ ] 页眉页脚正确（封面无、目录罗马数字、正文阿拉伯数字）
- [ ] 拼写和语法检查（`F7`）
- [ ] 图片已压缩（文件 → 压缩图片）
- [ ] 文档属性已填写（文件 → 信息 → 属性）

### 导出 PDF

!!! example "操作步骤"
    1. 文件 → 另存为 → 选择 PDF 格式
    2. 选项 → 勾选"创建书签时使用标题"
    3. 勾选"文档结构标记以便辅助功能"
    4. 保存

    导出的 PDF 会自动包含书签导航，方便阅读。
