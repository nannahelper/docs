# 第 4 章：长文档编排

> **从"散装页面"到"专业出版物"——驾驭 100 页以上的长文档**

论文、标书、手册、报告——当文档超过 20 页，手动维护目录、页码、交叉引用就变成了噩梦。本章教你用 Word 的自动化工具轻松管理长文档。

---

## 4.1 大纲视图与文档结构

### 大纲视图

**大纲视图** 是长文档编辑的核心工具，让你像操作文件管理器一样管理文档结构。

| 操作 | 快捷键 | 说明 |
|:---|:---|:---|
| 进入大纲视图 | `Ctrl+Alt+O` | 切换到大纲视图 |
| 提升标题级别 | `Shift+Tab` | 降级为正文 |
| 降低标题级别 | `Tab` | 升级标题级别 |
| 展开/折叠 | `Alt+Shift+加号/减号` | 展开或折叠子内容 |
| 移动段落 | `Alt+Shift+↑/↓` | 上下移动整个段落 |
| 仅显示首行 | `Alt+Shift+L` | 切换首行显示 |

!!! tip "大纲视图的威力"
    在大纲视图中，你可以：
    - **拖拽标题** 来重新排列整个章节
    - **折叠所有正文** 只显示标题，快速浏览文档结构
    - **一键提升/降低** 标题级别来调整层级

### 导航窗格

**导航窗格** （`Ctrl+F`）是页面视图下的"迷你大纲"，可以：
- 点击标题快速跳转到对应位置
- 拖拽标题重新排列章节
- 右键标题进行提升/降低/删除等操作

## 4.2 自动目录

### 目录的生成原理

Word 的自动目录基于 **标题样式** （标题 1、标题 2、标题 3 等）生成。只要你的标题应用了正确的标题样式，目录就能自动生成。

!!! warning "常见错误"
    很多人手动输入目录，然后每次修改文档后手动更新页码。这不仅低效，而且容易出错。 **永远使用自动目录。**

### 创建和自定义目录

!!! example "实战：创建三级目录"
    1. 确保所有章节标题应用了"标题 1"、"标题 2"、"标题 3"样式
    2. 将光标放在目录位置
    3. 引用 → 目录 → 自定义目录
    4. 显示级别：3
    5. 勾选"显示页码"和"页码右对齐"
    6. 制表符前导符：选择"……"

    修改文档后，右键目录 → 更新域 → 更新整个目录。

### 目录的高级定制

| 需求 | 方法 |
|:---|:---|
| 部分标题不显示在目录中 | 修改该标题样式 → 段落 → 大纲级别设为"正文文本" |
| 目录中显示非标题文字 | 选中文字 → 插入 → 书签 → 在目录中使用 TC 域 |
| 多个目录（如图目录、表目录） | 引用 → 插入表目录 → 题注标签选择"图"或"表" |
| 修改目录格式 | 引用 → 目录 → 自定义目录 → 修改 |

## 4.3 页眉页脚与页码

### 页眉页脚的核心概念

页眉页脚属于 **节** 的属性。通过分节符，你可以让不同节拥有不同的页眉页脚。

!!! info "链接到前一节"
    默认情况下，新节的页眉页脚会"链接到前一节"——即与上一节相同。取消这个链接后，当前节可以独立设置页眉页脚。

### 常见的页眉页脚方案

| 方案 | 实现方法 |
|:---|:---|
| **奇偶页不同** | 页眉页脚工具 → 勾选"奇偶页不同" |
| **首页不同** | 页眉页脚工具 → 勾选"首页不同" |
| **各章页眉不同** | 每章之间插入分节符 → 取消"链接到前一节" → 分别设置 |
| **页码从第 N 页开始** | 页码 → 设置页码格式 → 起始页码设为 N |

!!! example "实战：论文页眉页脚设置"
    1. 封面和目录不设页眉页脚（使用分节符隔离）
    2. 正文部分：奇数页页眉显示"第 X 章 章节标题"，偶数页显示"论文题目"
    3. 页码：目录用罗马数字（I, II, III），正文用阿拉伯数字（1, 2, 3）
    4. 实现：封面 → 分节符 → 目录（页码格式设为罗马数字）→ 分节符 → 正文（页码格式设为阿拉伯数字，起始页码 1）

## 4.4 交叉引用与题注

### 题注

**题注** 是给图表、表格、公式添加的自动编号标签。

!!! example "实战：为图片添加自动编号题注"
    1. 右键图片 → 插入题注
    2. 标签：选择"图"（如没有则新建标签）
    3. 位置：所选项目下方
    4. 编号：勾选"包含章节号"（需要标题样式支持）

    此后插入新图片时，题注编号自动更新。

### 交叉引用

**交叉引用** 让你在正文中引用题注、标题、页码等，当目标位置或编号变化时，交叉引用自动更新。

| 引用类型 | 引用内容 | 示例 |
|:---|:---|:---|
| 标题 | 标题文字、页码 | "详见第 3 章" |
| 题注 | 标签和编号 | "如图 2-5 所示" |
| 书签 | 书签文字、页码 | "参见附录 A" |
| 脚注/尾注 | 脚注编号 | "见脚注 3" |

!!! tip "交叉引用的优势"
    当你插入新图导致所有图片编号后移时，正文中的"如图 2-5 所示"会自动更新为"如图 2-6 所示"。全选文档后按 `F9` 刷新所有域。

## 4.5 主控文档与子文档

### 适用场景

当文档超过 100 页，或需要多人协作编写不同章节时，使用 **主控文档** 将文档拆分为多个子文档。

!!! example "实战：创建主控文档"
    1. 进入大纲视图
    2. 选中要拆分为子文档的标题和内容
    3. 点击"显示文档" → "创建"
    4. Word 自动将每个标题 1 拆分为独立的子文档
    5. 保存主控文档，子文档自动保存为独立文件

    每个子文档可以独立编辑，在主控文档中打开时自动合并显示。
