> **让学术写作变得优雅** —— 从零开始掌握 LaTeX 排版系统，写出专业美观的论文、报告和简历。

---

## 教程简介

LaTeX 是 Leslie Lamport 在 1980 年代开发的文档排版系统，底层基于 Donald Knuth 的 TeX 排版引擎。与 Word 等"所见即所得"软件不同，LaTeX 采用"所想即所得"的理念——你只需关注内容和结构，排版由系统自动完成。

!!! info "为什么学 LaTeX？"

    - **学术出版标准**：绝大多数 SCI 期刊和学术会议接受 LaTeX 格式投稿
    - **数学公式之王**：排版数学公式的能力无出其右
    - **自动化排版**：目录、参考文献、交叉引用全自动生成
    - **版本控制友好**：纯文本格式，可用 Git 管理论文版本

| 项目 | 内容 |
|:---|:---|
| **学习目标** | 掌握 LaTeX 基础语法，能独立撰写学术论文和报告 |
| **预计时长** | 4~6 小时 |
| **关键概念** | 文档类、导言区、环境、命令、数学模式、参考文献 |
| **实践任务** | 撰写一篇包含公式、图表和参考文献的学术短文 |

## 快速预览

```latex
\documentclass{article}
\usepackage[UTF8]{ctex}

\title{我的第一篇 LaTeX 文档}
\author{张三}
\date{2026 年 5 月}

\begin{document}
\maketitle

\section{引言}
这是引言部分。LaTeX 会自动处理段落缩进和间距。

\section{方法}
这里可以插入数学公式：
\begin{equation}
    E = mc^2
\end{equation}

\section{结论}
本文展示了 LaTeX 的基本用法。

\end{document}
```

**渲染效果：** 编译后生成 PDF 文件，包含自动居中的标题、作者和日期信息，章节自动编号，数学公式精美排版，整体呈现专业学术风格。
