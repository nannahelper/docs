# 第 3 章：常用技巧

> **让你的文档更专业** —— 表格、任务列表、脚注、HTML 混用等进阶技巧。

---

## 3.1 表格

```markdown
| 姓名 | 年龄 | 职业 |
|:---|:---:| ---:|
| 张三 | 25 | 工程师 |
| 李四 | 30 | 设计师 |
| 王五 | 28 | 产品经理 |
```

**渲染效果：** 表格有清晰的边框和行列结构。`:---` 表示左对齐，`:---:` 表示居中，`---:` 表示右对齐。

!!! tip "表格对齐"

    - `:---` → 左对齐（默认）
    - `:---:` → 居中对齐
    - `---:` → 右对齐

## 3.2 任务列表

```markdown
- [x] 已完成的任务
- [x] 也已完成
- [ ] 待完成的任务
- [ ] 另一个待办事项
```

**渲染效果：** 已完成的任务前面显示勾选的复选框（☑），未完成的显示空心复选框（☐）。在 GitHub 上可以直接点击切换状态。

## 3.3 脚注

```markdown
这是一段带有脚注的文字[^1]。

[^1]: 这是脚注的内容，会显示在页面底部。
```

**渲染效果：** 正文中脚注标记显示为上标数字或符号，点击可跳转到页面底部的脚注内容。脚注内容通常以较小字号显示。

## 3.4 转义字符

当你想显示 Markdown 的特殊符号本身（而非其标记含义）时，使用反斜杠 `\` 转义：

```markdown
\*这不是斜体\*

\# 这不是标题

\\ 这是反斜杠本身

\` 这不是代码
```

**渲染效果：** 转义后的符号按原样显示，不会被解析为 Markdown 标记。

## 3.5 HTML 混用

Markdown 支持直接嵌入 HTML 标签，实现更复杂的排版：

```markdown
这是 Markdown 文字，<span style="color: red;">这是红色文字</span>。

<details>
<summary>点击展开详情</summary>

这里是被折叠的内容，可以包含 Markdown 语法。

- 列表项 1
- 列表项 2

</details>

<div align="center">
  <img src="logo.png" width="200" alt="Logo">
  <p><em>居中的图片和说明</em></p>
</div>
```

**渲染效果：** HTML 标签会被浏览器正常渲染。`<details>` 创建可折叠区域，`<div align="center">` 实现内容居中。

## 3.6 Emoji 表情

```markdown
:smile: :heart: :rocket: :tada: :fire: :books:
```

**渲染效果：** 短代码被替换为对应的 emoji 图标：😄 ❤️ 🚀 🎉 🔥 📚

## 3.7 锚点链接

```markdown
[跳转到第三章](#3-常用技巧)

## 3. 常用技巧
```

**渲染效果：** 点击链接后页面平滑滚动到目标标题位置。锚点 ID 由标题自动生成（空格变 `-`，特殊字符被移除）。

## 3.8 Mermaid 图表

部分平台（如 GitHub、MkDocs）支持 Mermaid 语法绘制图表：

````markdown
​```mermaid
flowchart TD
    A[开始] --> B{判断条件}
    B -->|是| C[执行操作]
    B -->|否| D[结束]
    C --> D
​```
````

**渲染效果：** 代码块被渲染为可视化的流程图，包含矩形节点、菱形判断框和箭头连线。
