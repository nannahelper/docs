# Markdown 新手指南

> **让文档写作像聊天一样简单** —— 从零开始掌握 Markdown 标记语言，轻松写出结构清晰、排版美观的文档。

---

## 教程简介

Markdown 是一种轻量级标记语言，由 John Gruber 于 2004 年创建。它用简洁的纯文本符号来表达格式，让你专注于内容而非排版。无论是写技术文档、记笔记、发博客还是写 README，Markdown 都是现代开发者的必备技能。

!!! info "为什么学 Markdown？"

    - **极低的学习成本**：核心语法只需 10 分钟就能掌握
    - **跨平台通用**：GitHub、Notion、Obsidian、Typora 等几乎所有写作工具都支持
    - **纯文本格式**：可以用任何编辑器打开，版本控制友好
    - **导出灵活**：可轻松转换为 HTML、PDF、Word 等格式

| 项目 | 内容 |
|:---|:---|
| **学习目标** | 掌握 Markdown 核心语法，能独立撰写结构清晰的技术文档 |
| **预计时长** | 4~6 小时 |
| **关键概念** | 标题、段落、列表、链接、图片、代码块、表格、引用 |
| **实践任务** | 撰写一篇完整的项目 README 文档 |

## 这是二级标题

- 无序列表项 1
- 无序列表项 2

1. 有序列表项 1

[这是一个链接](https://example.com)

![图片描述](image.png)

`这是行内代码`

​```python
# 这是代码块
print("Hello, Markdown!")
​```
```

**渲染效果：** 上面的纯文本经过 Markdown 渲染后，会变成带有层级标题、格式化文字、列表、链接和语法高亮代码块的漂亮文档。
