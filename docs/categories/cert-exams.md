# 计算机等级考试

全国计算机等级考试（NCRE）三级选考项目教程。本分类包含三个选考科目的系统教程，以考试大纲为依据，用生活化比喻拆解抽象概念，帮助你高效备考。

---

## 可选科目

| 科目 | 简介 | 适合人群 |
|:---|:---|:---|
| [**三级网络技术**](../cert3-network/index.md) | 系统掌握计算机网络体系结构、协议原理和实用技术 | 对网络通信、Web 开发、运维部署感兴趣 |
| [**三级信息安全**](../cert3-security/index.md) | 从密码学到 Web 安全，覆盖信息安全全链条 | 对安全攻防、渗透测试、安全运维感兴趣 |
| [**三级嵌入式系统**](../cert3-embedded/index.md) | 软硬结合，掌握 ARM 架构、RTOS、接口技术 | 对物联网、智能硬件、驱动开发感兴趣 |

---

## 教程特色

-  **考试导向**：严格依据 NCRE 三级考试大纲，覆盖核心考点
-  **生活比喻**：用"快递物流"理解网络分层，用"家庭安防"理解信息安全，用"智能家居"理解嵌入式系统
-  **实战练习**：每章配有课后练习和真题演练
-  **模拟测试**：每科最后一章提供综合模拟测试，检验学习成果

---

## 如何选择科目？

| 你的兴趣/方向 | 推荐科目 |
|:---|:---|
| 想从事 Web 开发、运维、云计算 | 网络技术 |
| 想从事安全攻防、渗透测试 | 信息安全 |
| 想从事物联网、智能硬件、嵌入式开发 | 嵌入式系统 |
| 不确定方向，想打好基础 | 网络技术（知识最通用） |

---

## 学习建议

1. **按章节顺序学习**：每科 8 章，建议每天 1~2 章，1~2 周完成
2. **重视课后练习**：每章末尾的练习题覆盖了该章核心考点
3. **善用比喻理解**：遇到抽象概念时，回想对应的生活比喻
4. **模拟测试检验**：学完全部章节后，用第 8 章的模拟测试检验学习效果
---

## 重要提醒

!!! warning "备考建议"
    **计算机三级考试主要侧重于概念性知识的记忆与理解。** 为确保顺利通过考试，建议购买官方指定的考试指导用书，系统学习并重点记忆书中的细碎知识点和关键概念。
