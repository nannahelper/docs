# 基础技能

> **工欲善其事，必先利其器** —— 掌握软件开发的基础工具和文档写作方法。

---

## 分类说明

基础技能类教程聚焦于软件开发中的通用工具和方法论，不涉及特定编程语言或技术领域。无论你学习哪个方向，这些技能都是必备的基础。

| 方向 | 涵盖内容 |
|:---|:---|
| **开发工具** | 版本控制、IDE 使用、终端命令 |
| **文档写作** | Markdown、LaTeX、技术写作 |
| **环境配置** | 系统安装、环境搭建、部署运维 |

## 学习建议

建议按以下顺序学习基础技能类教程：

1. **Markdown 新手指南** — 先学会写文档，后续所有笔记和作业都用 Markdown
2. **Git 与 GitHub 团队协作指南** — 学会版本控制，管理你的代码和学习笔记
3. **MkDocs 与 GitHub Pages 部署指南** — 搭建个人文档网站，展示学习成果
4. **LaTeX 新手指南** — 需要写学术论文或精美简历时学习
