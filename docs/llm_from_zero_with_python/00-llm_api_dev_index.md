# 🚀 零基础 LLM API 开发入门：从安装到实战

欢迎来到本教程！这是一份专为**零基础初学者**设计的实战课程。

## 你将学到什么

完成本教程后，你将能够：

- ✅ 搭建完整的 Python 编程环境（VSCode + 虚拟环境）
- ✅ 通过代码调用 DeepSeek 大模型 API，告别手动复制粘贴
- ✅ 理解网页对话与 API 调用的本质区别
- ✅ 构建一个具备上下文记忆的终端 Chatbot
- ✅ 用 100 行 Python 代码自动清洗数千条非结构化数据并导出 Excel
- ✅ 掌握 API Key 安全管理，避免密钥泄露和意外扣费

## 目标受众

- 从未写过 Python 代码，但用过 ChatGPT / DeepSeek 网页版的同学
- 想用 AI 自动化重复性办公任务的职场人
- 对"用代码控制 AI"感到好奇的探索者
