# 第 7 章：局域网与无线网络

> **场景：** 你家里的路由器连着几台电脑、手机、智能电视——这就是一个典型的局域网。公司里几百台电脑通过交换机互联——这也是局域网。局域网是我们日常接触最多的网络形态。

---

## 7.1 以太网（Ethernet）

!!! example "核心比喻：以太网就像一个大办公室里的内部通信系统"
    同一个办公室里的同事交流不需要通过邮局——直接走过去说就行。以太网就是同一局域网内设备的"直接对话"机制。

### 以太网帧格式

```
┌──────┬──────┬──────┬──────┬──────┬──────┐
│前导码  │目的MAC│源MAC │类型/长度│ 数据  │ FCS  │
│ 8B   │ 6B   │ 6B   │ 2B    │46~1500B│ 4B  │
└──────┴──────┴──────┴──────┴──────┴──────┘
```

### 以太网标准演进

| 标准 | 速率 | 介质 | 年代 |
|:---|:---|:---|:---|
| 10BASE-T | 10 Mbps | 双绞线 | 1990 |
| 100BASE-TX（快速以太网） | 100 Mbps | 双绞线 | 1995 |
| 1000BASE-T（千兆以太网） | 1 Gbps | 双绞线 | 1999 |
| 10GBASE-T（万兆以太网） | 10 Gbps | 双绞线/光纤 | 2006 |
| 40G/100G 以太网 | 40/100 Gbps | 光纤 | 2010+ |

## 7.2 交换机与网桥

!!! example "核心比喻：交换机就是办公室里的智能邮件分拣员"
    集线器（Hub）是"大喇叭"——收到一封信，对着全办公室喊："张三的信！"所有人都听到了，只有张三来拿。
    
    交换机（Switch）是"智能分拣员"——它记住了每个人的座位（MAC 地址表），收到信后直接送到张三桌上，其他人不受干扰。

| 对比维度 | 集线器（Hub） | 交换机（Switch） |
|:---|:---|:---|
| 工作层次 | 物理层 | 数据链路层 |
| 转发方式 | 广播到所有端口 | 根据 MAC 地址表精确转发 |
| 冲突域 | 所有端口共享一个冲突域 | 每个端口独立冲突域 |
| 带宽利用 | 共享带宽 | 独享带宽 |
| 安全性 | 低（所有端口都能收到） | 高（只发给目标端口） |

---

## 7.3 VLAN —— 虚拟局域网

!!! example "核心比喻：VLAN 就是在开放办公区用隔板划分部门"
    一个大开间里坐着市场部、研发部、财务部。虽然物理上在同一空间，但用隔板隔开，各部门互不干扰。
    
    VLAN 在同一个物理交换机上划分出多个逻辑隔离的网络——市场部 VLAN 10，研发部 VLAN 20，财务部 VLAN 30。不同 VLAN 之间不能直接通信，必须通过路由器。

```
┌─────────────────────────────────────┐
│           同一台交换机                │
│  ┌─────────┐ ┌─────────┐ ┌────────┐│
│  │ VLAN 10 │ │ VLAN 20 │ │VLAN 30 ││
│  │ 市场部   │ │ 研发部   │ │ 财务部  ││
│  │192.168.1│ │192.168.2│ │192.168.3││
│  └─────────┘ └─────────┘ └────────┘│
└─────────────────────────────────────┘
```

| 概念 | 说明 |
|:---|:---|
| VLAN ID | 标识 VLAN 的编号，范围 1~4094 |
| Trunk 链路 | 承载多个 VLAN 流量的链路（使用 802.1Q 标签） |
| Access 链路 | 只属于一个 VLAN 的链路 |
| VLAN 间路由 | 不同 VLAN 通信需要通过路由器或三层交换机 |

## 7.4 无线局域网（WLAN / Wi-Fi）

### Wi-Fi 标准演进

| 标准 | 代号 | 频段 | 最大速率 | 年代 |
|:---|:---|:---|:---|:---|
| 802.11b | — | 2.4 GHz | 11 Mbps | 1999 |
| 802.11a | — | 5 GHz | 54 Mbps | 1999 |
| 802.11g | — | 2.4 GHz | 54 Mbps | 2003 |
| 802.11n | Wi-Fi 4 | 2.4/5 GHz | 600 Mbps | 2009 |
| 802.11ac | Wi-Fi 5 | 5 GHz | 6.9 Gbps | 2013 |
| 802.11ax | Wi-Fi 6 | 2.4/5/6 GHz | 9.6 Gbps | 2019 |

### CSMA/CA —— 无线网络的"避让规则"

无线网络不能使用 CSMA/CD（冲突检测），因为无线信号在发送时会淹没接收信号。所以使用 CSMA/CA（冲突避免）：

1. 发送前先监听信道
2. 信道空闲后等待一段随机时间
3. 发送前先发一个短请求帧（RTS）
4. 收到允许发送帧（CTS）后才开始发送
5. 收到确认帧（ACK）才算发送成功

## 7.5 常见题型

**例题 1：** 交换机工作在 OSI 模型的（ ）。

A. 物理层　　B. 数据链路层　　C. 网络层　　D. 传输层

??? answer "查看答案"
    **答案：B**
    
    交换机根据 MAC 地址进行帧的转发，工作在数据链路层。集线器工作在物理层，路由器工作在网络层。

**例题 2：** VLAN 之间的通信必须通过（ ）。

A. 集线器　　B. 二层交换机　　C. 路由器或三层交换机　　D. 中继器

??? answer "查看答案"
    **答案：C**
    
    不同 VLAN 属于不同的广播域（即不同的 IP 网段），它们之间的通信需要网络层的路由功能，因此必须通过路由器或具有路由功能的三层交换机。

**例题 3：** 802.11 无线局域网使用的介质访问控制方法是（ ）。

A. CSMA/CD　　B. 令牌传递　　C. CSMA/CA　　D. TDMA

??? answer "查看答案"
    **答案：C**
    
    无线局域网使用 CSMA/CA（载波监听多路访问/冲突避免），因为无线设备无法在发送的同时检测冲突。

## 要点总结

- [x] 以太网是使用最广泛的局域网技术，速率从 10Mbps 到 100Gbps+
- [x] 交换机根据 MAC 地址表精确转发，每个端口独享带宽
- [x] VLAN 在物理交换机上划分逻辑隔离的网络，VLAN 间通信需要路由
- [x] Wi-Fi 标准从 802.11b 演进到 Wi-Fi 6（802.11ax）
- [x] 无线网络使用 CSMA/CA 而非 CSMA/CD
