## 📖 各章速览

### 第1章：认识Linux
**核心比喻**：遥控器 vs 语音控制

- Windows/Mac = 带按钮的遥控器  
- Linux = 语音控制的智能助手  
- **关键命令**：`ls`（看看房间里有什么）

### 第2章：搭建环境  
**核心比喻**：租房 vs 买房  

- 虚拟机 = 在现有系统里”租个房间”  
- 云服务器 = 在云端“租个公寓”  
- **实践任务**：安装Ubuntu到VirtualBox

### 第3章：命令初体验
**核心比喻**：语音遥控器  

- 终端 = 麦克风  
- 命令 = 你说的指令  
- **四大生存命令**：`pwd`, `ls`, `cd`, `clear`

### 第4章：文件操作
**核心比喻**：房间和抽屉  

- 根目录`/` = 房子的地基  
- `home`目录 = 你的卧室  
- **核心操作**：创建、删除、复制、移动、查看

### 第5章：用户权限
**核心比喻**：钥匙和锁  

- 普通用户 = 房客  
- root用户 = 房东  
- sudo = 临时万能钥匙  
- **安全原则**：最小权限，谨慎使用sudo

### 第6章：实用技巧
**核心比喻**：智能管家  

- Tab补全 = 智能输入提示  
- 管道`|` = 流水线传送带  
- 脚本 = 管家任务清单  
- **下一步**：Shell脚本、服务器管理、容器技术

## 📝 学习建议

1. **按顺序学习** — 比喻体系是连贯的
2. **动手实践** — 每个命令都要自己敲一遍
3. **不怕犯错** — Linux很宽容，虚拟机里随便试
4. **复习比喻** — 忘记概念时回想对应的生活场景
