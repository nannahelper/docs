## 关键公式与法则

| 法则 | 表述 |
|------|------|
| **Brooks 法则** | 向进度落后的项目加人，只会使进度更落后 |
| **沟通路径公式** | n(n-1)/2 |
| **规模-工作量关系** | 工作量 ≈ 常数 × (规模)^1.5 |
| **编程系统产品成本** | 独立程序 × 9 |
| **进度黄金分配** | 1/3 计划 + 1/6 编码 + 1/4 构件测试 + 1/4 系统测试 |
| **维护成本** | ≥ 开发成本的 40% |
| **缺陷修复引入率** | 20-50% 概率引入新 bug |
| **根本困难** | 复杂度、一致性、可变性、不可见性 |
