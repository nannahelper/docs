## 教程简介

R 语言是由新西兰统计学家 Ross Ihaka 和 Robert Gentleman 于 1993 年开发的统计计算与图形绘制语言。它是数据科学领域的"瑞士军刀"，在学术研究、金融分析和生物信息学中广泛应用。

!!! info "为什么学 R 语言？"

    - **统计计算王者**：内置海量统计函数，从 t 检验到时间序列分析一应俱全
    - **数据可视化标杆**：ggplot2 包被誉为"图形语法"的完美实现
    - **学术研究标配**：社会科学、生物统计、计量经济学的首选工具
    - **活跃的社区**：CRAN 上有 20,000+ 扩展包，几乎覆盖所有分析需求

## 快速预览

```r
# 创建数据
students <- data.frame(
    name = c("张三", "李四", "王五", "赵六"),
    score = c(85, 92, 78, 88),
    grade = c("B", "A", "C", "B")
)

# 查看数据
print(students)

# 基本统计
mean(students$score)    # 平均分
summary(students$score) # 描述性统计

# 可视化
library(ggplot2)
ggplot(students, aes(x = name, y = score, fill = name)) +
    geom_bar(stat = "identity") +
    labs(title = "学生成绩对比", x = "姓名", y = "分数") +
    theme_minimal()
```

**渲染效果：** 代码运行后在 RStudio 中输出数据表格、统计摘要和一张彩色柱状图。图表有标题、坐标轴标签和简洁的主题样式。
