# 第 6 章：演讲者工具与演示技巧

> **从"手忙脚乱"到"游刃有余"——掌握演讲者工具，自信登台**

一份好的 PPT 只是成功的一半，另一半在于如何呈现它。本章教你使用 PowerPoint 的演讲者工具和演示技巧，让你在任何场合都能自信表达。

---

## 6.1 演讲者视图

### 什么是演讲者视图？

**演讲者视图** 让你在自己的屏幕上看到备注、下一张幻灯片预览、计时器和工具栏，而观众只能看到当前幻灯片。

!!! example "实战：启用演讲者视图"
    1. 连接投影仪或外接显示器
    2. 幻灯片放映 → 勾选"使用演讲者视图"
    3. 开始放映（`F5`）

    你的屏幕显示：
    - 左上：当前幻灯片
    - 右上：下一张幻灯片预览
    - 下方：备注区域
    - 底部：计时器、画笔工具、幻灯片缩略图

### 演讲者视图的布局

```
┌─────────────────────────────────────────┐
│  ┌──────────────┐  ┌──────────────────┐ │
│  │              │  │                  │ │
│  │  当前幻灯片   │  │  下一张幻灯片     │ │
│  │              │  │  （预览）         │ │
│  └──────────────┘  └──────────────────┘ │
│  ┌─────────────────────────────────────┐ │
│  │  备注：                              │ │
│  │  这里是只有你能看到的演讲备注...       │ │
│  │                                      │ │
│  └─────────────────────────────────────┘ │
│  ⏱ 00:15:32  ▶ 第 5/20 页  🖊 笔  🔍   │
└─────────────────────────────────────────┘
```

## 6.2 演讲备注

### 备注的用途

| 用途 | 示例 |
|:---|:---|
| **关键数据** | "Q3 增长 23%，行业平均 12%" |
| **过渡语句** | "接下来我们看看市场反馈..." |
| **提醒事项** | "此处停顿 3 秒，等待观众反应" |
| **备用案例** | "如果被问到竞品对比，补充 XX 数据" |

!!! tip "备注 vs 幻灯片内容"
    幻灯片上放 **结论**，备注里放 **论据**。观众看到的是"销售额增长 23%"，你看到的是"Q1 增长 15%，Q2 增长 18%，Q3 增长 23%，主要驱动因素是新产品上市和渠道扩张"。

### 备注页视图

视图 → 备注页，可以看到每张幻灯片及其备注的打印预览效果。适合打印出来作为演讲手卡。

## 6.3 演示快捷键

### 必备快捷键

| 快捷键 | 功能 |
|:---|:---|
| `F5` | 从头开始放映 |
| `Shift+F5` | 从当前幻灯片开始放映 |
| `N` / `→` / `空格` | 下一张 |
| `P` / `←` / `Backspace` | 上一张 |
| `数字 + Enter` | 跳转到指定编号的幻灯片 |
| `B` | 黑屏（再按恢复） |
| `W` | 白屏（再按恢复） |
| `Ctrl+P` | 切换为笔（可在幻灯片上标注） |
| `Ctrl+A` | 切换为箭头 |
| `Ctrl+E` | 切换为橡皮擦 |
| `E` | 清除所有墨迹 |
| `Ctrl+T` | 显示任务栏 |
| `Esc` | 结束放映 |

!!! tip "B/W 键的妙用"
    演示中按 `B` 键黑屏，让观众注意力回到你身上。这在问答环节或需要强调口头表达时非常有用。

## 6.4 排练计时

### 录制排练

!!! example "实战：使用排练计时"
    1. 幻灯片放映 → 排练计时
    2. 像正式演讲一样完整演示一遍
    3. 每页的停留时间自动记录
    4. 结束时选择"保留"计时

    此后放映时可以选择"使用计时"自动翻页，或查看每页的建议时长。

### 排练报告

排练结束后，查看每张幻灯片的用时分布，识别：
- 哪些页面讲得太快（可能需要补充内容）
- 哪些页面讲得太慢（可能需要精简）
- 总时长是否符合要求

## 6.5 演示技巧

### 演示前检查清单

| 检查项 | 说明 |
|:---|:---|
| 字体嵌入 | 文件 → 选项 → 保存 → 嵌入字体 |
| 视频/音频 | 确认媒体文件能正常播放 |
| 投影比例 | 确认投影仪是 16:9 还是 4:3 |
| 颜色显示 | 在投影仪上颜色可能与屏幕不同 |
| 备用方案 | 导出 PDF 版本以防万一 |

### 演示中的技巧

| 技巧 | 说明 |
|:---|:---|
| **面向观众** | 不要对着屏幕读幻灯片 |
| **激光笔** | 按住 `Ctrl` + 鼠标左键模拟激光笔 |
| **放大细节** | 放映中 `+` 键放大，`-` 键缩小 |
| **快速定位** | 右键 → 查看所有幻灯片 → 点击跳转 |
| **隐藏幻灯片** | 右键幻灯片 → 隐藏（不删除，放映时跳过） |

## 6.6 演示模式

### 三种演示模式

| 模式 | 用途 | 启动方式 |
|:---|:---|:---|
| **演讲者模式** | 双屏演示（推荐） | 幻灯片放映 → 演讲者视图 |
| **观众自行浏览** | 展台、自助浏览 | 幻灯片放映 → 设置幻灯片放映 → 观众自行浏览 |
| **展台模式** | 自动循环播放 | 幻灯片放映 → 设置幻灯片放映 → 在展台浏览 |

!!! example "实战：设置展台自动循环播放"
    1. 幻灯片放映 → 设置幻灯片放映
    2. 放映类型：在展台浏览（全屏幕）
    3. 放映选项：循环放映，按 Esc 键终止
    4. 换片方式：如果存在排练时间，则使用它

    适合展会、前台、休息区等无人值守场景。
