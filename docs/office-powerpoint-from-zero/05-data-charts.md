# 第 5 章：数据图表展示

> **从"粘贴 Excel 截图"到"让数据讲故事"——在 PPT 中展示数据的最佳实践**

大多数人在 PPT 中展示数据的方式是：从 Excel 截图，粘贴到 PPT，缩小到看不清。本章教你如何让数据在演示中清晰、有力、有说服力。

---

## 5.1 在 PPT 中创建图表

### 插入图表

插入 → 图表 → 选择图表类型。PPT 会打开一个嵌入的 Excel 窗口供你编辑数据。

| 图表类型 | PPT 中的最佳用途 |
|:---|:---|
| **柱形图** | 类别对比（不超过 7 个类别） |
| **条形图** | 排名展示（尤其是长标签） |
| **折线图** | 趋势展示（时间序列） |
| **饼图** | 占比展示（不超过 5 个部分） |
| **面积图** | 量级趋势 |
| **散点图** | 相关性分析 |

### 图表链接 vs 嵌入

| 方式 | 操作 | 优点 | 缺点 |
|:---|:---|:---|:---|
| **嵌入** | 直接在 PPT 中创建 | 独立、便携 | 数据不联动 |
| **链接** | 从 Excel 复制 → 粘贴 → 保留源格式和链接数据 | 数据更新自动同步 | 需要 Excel 文件 |

!!! example "实战：链接 Excel 图表到 PPT"
    1. 在 Excel 中选中图表 → `Ctrl+C`
    2. 在 PPT 中 → 开始 → 粘贴 → 选择性粘贴
    3. 选择"粘贴链接" → Microsoft Excel 图表对象
    4. 确定

    此后 Excel 中数据更新，PPT 中图表右键 → 更新链接即可同步。

## 5.2 图表动画

### 按系列/按类别动画

!!! example "实战：柱形图逐系列显示"
    1. 选中图表 → 动画 → 添加动画 → 进入 → 擦除
    2. 效果选项 → 方向：自底部
    3. 效果选项 → 按系列
    4. 动画窗格中展开 → 调整各系列的延迟

    效果：单击一次，第一个系列从底部升起；再单击，第二个系列升起。

### 图表动画类型

| 动画 | 适合图表类型 | 效果 |
|:---|:---|:---|
| **擦除** | 柱形图、条形图 | 从底部/左侧生长 |
| **淡入** | 所有类型 | 柔和出现 |
| **缩放** | 饼图 | 从中心展开 |
| **飞入** | 所有类型 | 从指定方向飞入 |

## 5.3 表格美化

### 表格设计原则

| 原则 | 说明 |
|:---|:---|
| **去除冗余** | 删除不必要的边框和填充 |
| **对齐数字** | 数字右对齐，文字左对齐 |
| **表头突出** | 表头使用深色背景 + 白色文字 |
| **斑马条纹** | 隔行使用浅色背景增强可读性 |
| **高亮重点** | 用强调色标注关键数据 |

!!! example "实战：将默认表格改造为专业表格"
    **改造步骤：**
    1. 选中表格 → 表格设计 → 取消"标题行"以外的所有选项
    2. 取消所有边框（表格设计 → 边框 → 无边框）
    3. 表头：深蓝背景 + 白色加粗文字
    4. 数据行：隔行添加浅灰背景
    5. 数字列：右对齐
    6. 底部添加汇总行：加粗 + 上边框线

## 5.4 信息图设计

### 用形状制作信息图

不需要专业设计软件，PowerPoint 的形状工具就能制作信息图。

!!! example "实战：制作"增长指标"信息图"
    1. 插入 → 形状 → 圆形（大圆作为背景）
    2. 插入 → 形状 → 弧形（调整为半圆弧，作为进度环）
    3. 插入 → 文本框 → 输入百分比数字（大号加粗）
    4. 插入 → 文本框 → 输入指标名称（小号）
    5. 组合所有元素（`Ctrl+G`）
    6. 复制三份，修改数字和颜色

### 常用信息图元素

| 元素 | 制作方法 |
|:---|:---|
| **进度条** | 圆角矩形 + 填充矩形（不同颜色） |
| **环形图** | 插入 → 图表 → 环形图（只保留一个数据点） |
| **图标列表** | 图标 + 文字，使用对齐工具排列 |
| **时间线** | 直线 + 圆形节点 + 文字标签 |
| **漏斗图** | 插入 → SmartArt → 漏斗 |
