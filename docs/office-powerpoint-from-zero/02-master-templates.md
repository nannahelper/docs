# 第 2 章：母版与模板

> **从"逐页调整"到"一次设计，全局应用"——母版是 PPT 效率的核武器**

如果你还在逐页复制粘贴 Logo、逐页调整标题位置，那么你浪费了 80% 的排版时间。母版让你一次设计，所有页面自动继承。

---

## 2.1 母版视图

### 母版的层级结构

```
主题（Theme）
  └── 幻灯片母版（Slide Master）—— 控制所有版式的公共元素
        ├── 版式 1：标题幻灯片
        ├── 版式 2：标题和内容
        ├── 版式 3：节标题
        ├── 版式 4：两栏内容
        └── ... 更多版式
```

!!! info "母版 vs 版式"
    - **幻灯片母版**（最大的那张）：修改它会影响 **所有版式**
    - **版式**（下面的小图）：修改它只影响使用该版式的幻灯片

### 进入母版视图

视图 → 幻灯片母版（或 `Alt+W+M`）

---

## 2.2 母版设计实战

### 在母版中添加公共元素

!!! example "实战：为所有幻灯片添加公司 Logo 和页码"
    1. 视图 → 幻灯片母版
    2. 选中 **最大的那张母版**（不是下面的版式）
    3. 插入 → 图片 → 选择公司 Logo
    4. 将 Logo 拖到右上角，调整大小
    5. 插入 → 页眉和页脚 → 勾选"幻灯片编号" → 全部应用
    6. 调整页码的位置和格式
    7. 关闭母版视图

    现在所有幻灯片自动带有 Logo 和页码，新增幻灯片也会自动继承。

### 自定义版式

!!! example "实战：创建"案例研究"专用版式"
    1. 视图 → 幻灯片母版
    2. 在左侧缩略图中右键 → 插入版式
    3. 重命名为"案例研究"
    4. 插入占位符：
       - 标题占位符（顶部）
       - 文本占位符（左侧，用于案例描述）
       - 图片占位符（右侧，用于案例配图）
       - 文本占位符（底部，用于关键结论）
    5. 设置各占位符的字体、颜色、大小
    6. 关闭母版视图

    此后新建幻灯片时，在版式列表中就能看到"案例研究"版式。

### 占位符类型

| 占位符类型 | 用途 |
|:---|:---|
| **内容** | 通用占位符（可插入文字、图片、图表等） |
| **文字** | 纯文本占位符 |
| **图片** | 图片占位符（点击即可选图） |
| **图表** | 图表占位符 |
| **表格** | 表格占位符 |
| **SmartArt** | SmartArt 图形占位符 |
| **媒体** | 视频/音频占位符 |

## 2.3 多母版管理

### 使用场景

一份演示文稿可以包含多个母版，每个母版对应不同的视觉风格。

| 场景 | 母版方案 |
|:---|:---|
| 同一公司不同子品牌 | 每个品牌一个母版 |
| 同一报告不同章节 | 每章一个母版（不同配色） |
| 主报告 + 附录 | 主报告母版 + 附录母版（简洁版） |

!!! example "实战：添加第二个母版"
    1. 视图 → 幻灯片母版
    2. 在左侧缩略图最下方空白处右键 → 插入幻灯片母版
    3. 设计第二个母版的样式
    4. 关闭母版视图
    5. 选中需要切换母版的幻灯片 → 开始 → 版式 → 选择第二个母版下的版式

## 2.4 主题与模板

### 主题、模板、母版的关系

| 概念 | 文件格式 | 包含内容 | 比喻 |
|:---|:---|:---|:---|
| **主题** | .thmx | 配色、字体、效果 | 装修风格方案 |
| **模板** | .potx | 母版 + 版式 + 示例幻灯片 | 精装样板间 |
| **演示文稿** | .pptx | 模板 + 实际内容 | 入住后的家 |

### 创建自定义模板

!!! example "实战：创建公司演示模板"
    1. 设计好母版（Logo、配色、字体、版式）
    2. 添加常用示例幻灯片（封面、目录、章节页、正文、结束页）
    3. 文件 → 另存为 → 文件类型选择"PowerPoint 模板（*.potx）"
    4. 保存到 `%USERPROFILE%\Documents\自定义 Office 模板`

    此后新建 → 个人 → 选择该模板，所有母版和示例幻灯片自动加载。

### 使用现有主题

| 操作 | 路径 |
|:---|:---|
| 应用内置主题 | 设计 → 主题 → 选择 |
| 浏览自定义主题 | 设计 → 主题 → 浏览主题 |
| 仅应用配色方案 | 设计 → 变体 → 颜色 |
| 仅应用字体方案 | 设计 → 变体 → 字体 |
