# 第 3 章：图形与多媒体

> **从"纯文字"到"视觉盛宴"——用图形和多媒体增强信息传达**

文字只能传达信息，图形和多媒体能创造体验。本章教你如何用 PowerPoint 内置的图形工具和多媒体功能，让演示文稿"活"起来。

---

## 3.1 SmartArt：概念可视化

### SmartArt 的八种类型

| 类型 | 用途 | 典型场景 |
|:---|:---|:---|
| **列表** | 展示非顺序信息 | 议程、要点总结 |
| **流程** | 展示步骤或时间线 | 项目计划、操作步骤 |
| **循环** | 展示循环往复的过程 | PDCA、生命周期 |
| **层次结构** | 展示层级关系 | 组织架构、决策树 |
| **关系** | 展示元素之间的关联 | 韦恩图、对比图 |
| **矩阵** | 展示四象限关系 | SWOT 分析、优先级矩阵 |
| **棱锥图** | 展示比例或层级关系 | 需求层次、漏斗分析 |
| **图片** | 图文结合的展示 | 团队介绍、产品展示 |

!!! example "实战：将文字列表转换为 SmartArt"
    1. 选中文本框中的列表文字
    2. 开始 → 转换为 SmartArt → 选择合适的图形
    3. 使用"SmartArt 设计"选项卡调整颜色和样式
    4. 使用"格式"选项卡微调形状效果

    5 秒将枯燥的列表变成专业的可视化图形。

### SmartArt 的设计技巧

| 技巧 | 说明 |
|:---|:---|
| **控制节点数量** | 每个 SmartArt 不超过 5~7 个节点 |
| **保持文字简洁** | 每个节点 3~8 个字 |
| **统一视觉风格** | 同一演示中使用相同的 SmartArt 样式 |
| **善用颜色** | 用强调色突出最重要的节点 |

## 3.2 图标与 SVG

### 内置图标库

PowerPoint 内置了数千个高质量图标，无需上网搜索。

!!! example "实战：使用内置图标"
    1. 插入 → 图标
    2. 搜索关键词（如"增长"、"团队"、"目标"）
    3. 选择一个或多个图标 → 插入
    4. 选中图标 → 图形格式 → 图形填充（更改颜色）
    5. 图标可以像形状一样缩放、旋转、添加效果

!!! tip "图标使用原则"
    - 同一演示中使用 **同一风格的图标**（线性/填充）
    - 图标颜色与配色方案一致
    - 图标 + 简短文字 > 纯文字 > 纯图标

### SVG 编辑

PowerPoint 支持 SVG 格式，插入后可以转换为形状进行编辑。

!!! example "实战：编辑 SVG 颜色"
    1. 插入 → 图片 → 选择 SVG 文件
    2. 选中 SVG → 图形格式 → 转换为形状
    3. 现在可以单独选中 SVG 的每个部分，分别修改颜色

## 3.3 图片处理

### 图片编辑工具

| 工具 | 用途 | 路径 |
|:---|:---|:---|
| **删除背景** | 去除图片背景 | 图片格式 → 删除背景 |
| **裁剪** | 裁剪为形状或比例 | 图片格式 → 裁剪 |
| **艺术效果** | 添加滤镜效果 | 图片格式 → 艺术效果 |
| **更正** | 调整亮度/对比度 | 图片格式 → 更正 |
| **颜色** | 重新着色 | 图片格式 → 颜色 |
| **压缩图片** | 减小文件体积 | 图片格式 → 压缩图片 |

!!! example "实战：制作圆形头像"
    1. 插入人物照片
    2. 图片格式 → 裁剪 → 纵横比 → 1:1（正方形）
    3. 图片格式 → 裁剪 → 裁剪为形状 → 圆形
    4. 添加白色边框（图片格式 → 图片边框 → 白色 → 3 磅）

!!! warning "图片压缩"
    高分辨率图片是 PPT 文件体积过大的主要原因。文件 → 另存为 → 工具 → 压缩图片 → 选择"电子邮件（96 ppi）"可大幅减小文件。

## 3.4 视频与音频

### 嵌入视频

| 来源 | 操作 |
|:---|:---|
| **本地文件** | 插入 → 视频 → 此设备 |
| **在线视频** | 插入 → 视频 → 联机视频（粘贴 YouTube/Vimeo 链接） |
| **屏幕录制** | 插入 → 屏幕录制 |

### 视频播放设置

| 设置 | 说明 |
|:---|:---|
| **自动播放** | 切换到该幻灯片时自动开始 |
| **单击播放** | 单击视频才开始播放 |
| **循环播放** | 播放完毕后自动重播 |
| **全屏播放** | 播放时自动全屏 |
| **剪裁视频** | 只播放视频的一部分 |

!!! example "实战：设置视频自动播放并添加淡入效果"
    1. 选中视频
    2. 播放 → 开始 → 自动
    3. 勾选"循环播放，直到停止"
    4. 视频格式 → 视频边框 → 添加圆角边框
    5. 动画 → 添加淡入效果

### 音频使用

| 用途 | 操作 |
|:---|:---|
| **背景音乐** | 插入 → 音频 → 此设备 → 播放 → 跨幻灯片播放 |
| **旁白录制** | 插入 → 音频 → 录制音频 |
| **音效** | 插入 → 音频 → 此设备 → 播放 → 单击播放 |

!!! tip "背景音乐最佳实践"
    - 音量不要盖过演讲者声音（建议设为 20%~30%）
    - 选择纯音乐或轻音乐，避免有歌词的音乐分散注意力
    - 在"播放"选项卡中勾选"跨幻灯片播放"和"循环播放"

## 3.5 形状与合并

### 形状合并的五种方式

| 合并方式 | 效果 | 用途 |
|:---|:---|:---|
| **联合** | 合并为一个形状 | 创建复杂轮廓 |
| **组合** | 合并但去除重叠部分 | 创建镂空效果 |
| **拆分** | 按重叠边界分割 | 创建拼图效果 |
| **相交** | 保留重叠部分 | 创建遮罩效果 |
| **剪除** | 从第一个形状减去重叠部分 | 创建缺口效果 |

!!! example "实战：创建文字镂空效果"
    1. 插入矩形（覆盖整个幻灯片）
    2. 插入文本框，输入文字
    3. 先选中矩形，再选中文本框
    4. 形状格式 → 合并形状 → 剪除
    
    文字区域被"挖空"，露出幻灯片背景。
