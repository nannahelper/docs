# 第 7 章：协作与共享

> **从"一个人做 PPT"到"团队高效协作"——共享、审阅与分发**

现代办公中，PPT 往往不是一个人完成的。本章教你如何与团队高效协作，以及如何将演示文稿以最佳方式分发给不同受众。

---

## 7.1 共同创作

### 实时协作

将文件保存到 OneDrive 或 SharePoint 后，多人可以同时编辑同一份演示文稿。

!!! example "实战：开启实时协作"
    1. 文件 → 共享 → 保存到 OneDrive
    2. 点击"共享"按钮
    3. 输入同事的邮箱地址
    4. 设置权限（可编辑 / 仅查看）
    5. 发送邀请

    协作者的头像会显示在右上角，他们的修改会实时同步，不同人的编辑位置用不同颜色标记。

### 协作注意事项

| 注意点 | 说明 |
|:---|:---|
| **避免冲突** | 不要同时编辑同一张幻灯片 |
| **使用批注** | 有问题用批注沟通，不要直接修改 |
| **版本历史** | 文件 → 信息 → 版本历史，可恢复任意历史版本 |
| **保存频率** | OneDrive 自动保存，但建议手动 `Ctrl+S` 确认 |

## 7.2 批注与审阅

### 批注

| 操作 | 路径 |
|:---|:---|
| 添加批注 | 审阅 → 新建批注（或 `Ctrl+Alt+M`） |
| 回复批注 | 点击批注 → 输入回复 |
| 解决批注 | 点击批注 → 解决（批注保留但标记为已处理） |
| 删除批注 | 右键批注 → 删除 |

!!! tip "批注的最佳实践"
    - 批注要具体：不要说"这里改一下"，要说"标题建议改为'Q3 业绩回顾'"
    - 解决后不要删除批注——保留讨论记录便于追溯
    - 使用 @提及 通知特定同事

### 比较文档

审阅 → 比较，可以比较两个版本的演示文稿，查看所有差异。

!!! example "实战：比较修改前后的版本"
    1. 审阅 → 比较
    2. 选择修改前的版本
    3. PowerPoint 显示所有差异：
       - 哪些幻灯片被添加/删除
       - 哪些内容被修改
    4. 逐条接受或拒绝修改

## 7.3 导出选项

### 导出格式对比

| 格式 | 用途 | 特点 |
|:---|:---|:---|
| **PDF** | 分发阅读版 | 保留版式，不可编辑，文件小 |
| **XPS** | 类似 PDF | Microsoft 的 PDF 替代方案 |
| **MP4 视频** | 自动播放演示 | 包含所有动画、过渡和计时 |
| **GIF 动图** | 社交媒体分享 | 将幻灯片导出为动图 |
| **图片（PNG/JPEG）** | 单页分享 | 每张幻灯片导出为一张图片 |
| **大纲/RTF** | 提取文字内容 | 只导出文本，不含格式 |
| **讲义** | 打印分发 | 一页多张幻灯片 + 备注行 |

!!! example "实战：导出为视频"
    1. 文件 → 导出 → 创建视频
    2. 选择分辨率：1080p（全高清）
    3. 选择是否使用录制的计时和旁白
    4. 设置每张幻灯片的秒数（默认 5 秒）
    5. 创建视频

    适合发送给无法参加现场演示的人，或上传到视频平台。

### 导出为 PDF 的最佳设置

!!! example "实战：导出高质量 PDF"
    1. 文件 → 另存为 → PDF
    2. 选项 → 发布选项：
       - 发布内容：幻灯片
       - 每页幻灯片数：1
       - 勾选"包括隐藏的幻灯片"（如需要）
    3. 工具 → 压缩图片 → 选择"打印（220 ppi）"

## 7.4 打包演示文稿

### 打包成 CD

将演示文稿和所有链接文件（视频、音频、字体）打包到一个文件夹。

!!! example "实战：打包演示文稿"
    1. 文件 → 导出 → 将演示文稿打包成 CD
    2. 添加要打包的文件（可添加多个）
    3. 选项 → 勾选：
       - 链接的文件
       - 嵌入的 TrueType 字体
    4. 复制到文件夹 → 选择目标位置

    打包后的文件夹包含所有必要文件，可以在没有 PowerPoint 的电脑上使用 PowerPoint Viewer 播放。

### 检查兼容性

文件 → 信息 → 检查问题 → 检查兼容性，查看演示文稿在旧版本 PowerPoint 中可能存在的问题。

| 常见兼容性问题 | 影响 |
|:---|:---|
| SmartArt 图形 | 在 PowerPoint 2007 之前显示为图片 |
| 某些切换效果 | 在旧版本中降级为淡入淡出 |
| 嵌入视频 | 旧版本可能无法播放 |
| SVG 图形 | PowerPoint 2016 之前不支持 |

## 7.5 保护演示文稿

### 保护选项

| 保护方式 | 路径 | 效果 |
|:---|:---|:---|
| **标记为最终** | 文件 → 信息 → 保护演示文稿 | 设为只读，防止编辑 |
| **密码加密** | 文件 → 信息 → 保护演示文稿 → 用密码加密 | 需要密码才能打开 |
| **限制编辑** | 审阅 → 限制编辑 | 限制格式修改 |
| **添加数字签名** | 文件 → 信息 → 保护演示文稿 → 添加数字签名 | 证明文件来源 |
