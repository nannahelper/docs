# 第 1 章：幻灯片基础与设计原则

> **从"文字堆砌"到"视觉叙事"——建立专业的幻灯片设计思维**

大多数人的 PPT 问题是：把 Word 文档复制粘贴到了 PowerPoint 里。本章帮你建立正确的幻灯片设计思维，让每一页都服务于你的核心信息。

---

## 1.1 幻灯片设计的四个核心原则

### CRAP 原则

| 原则 | 英文 | 含义 | 反面示例 |
|:---|:---|:---|:---|
| **对比** | Contrast | 通过差异引导注意力 | 所有文字一样大，分不清主次 |
| **重复** | Repetition | 统一元素增强专业感 | 每页字体颜色都不一样 |
| **对齐** | Alignment | 元素之间有视觉联系 | 文字图片随意摆放 |
| **亲密性** | Proximity | 相关内容靠近，不相关内容远离 | 标题和正文之间没有关联感 |

!!! example "实战：用 CRAP 原则改造一页幻灯片"
    **改造前：**
    - 标题 28 号字，正文 24 号字（对比不足）
    - 三个要点散乱排列（缺乏对齐）
    - 标题和正文间距与正文行间距相同（缺乏亲密性）
    
    **改造后：**
    - 标题 36 号加粗，正文 18 号（强对比）
    - 三个要点左对齐，使用统一的项目符号（对齐 + 重复）
    - 标题与正文间距 2 倍于正文行间距（亲密性）

### 6×6 法则

每张幻灯片不超过 **6 个项目符号** ，每个项目符号不超过 **6 个词** 。

!!! tip "这不是硬性规定，而是指导原则"
    6×6 法则的核心思想是： **幻灯片是演讲的视觉辅助，不是讲稿的替代品** 。观众应该听你讲，而不是读你的幻灯片。

## 1.2 版式系统

### 内置版式类型

| 版式 | 用途 | 典型场景 |
|:---|:---|:---|
| **标题幻灯片** | 演示封面 | 报告标题、演讲人、日期 |
| **标题和内容** | 最常用的正文版式 | 要点罗列、图文混排 |
| **节标题** | 章节过渡页 | 分隔演示的不同部分 |
| **两栏内容** | 左右对比 | 优缺点对比、前后对比 |
| **比较** | 带标题的两栏对比 | 方案 A vs 方案 B |
| **仅标题** | 自由布局 | 全屏图片、自定义设计 |
| **空白** | 完全自由设计 | 创意排版、信息图 |
| **内容与标题** | 标题在左，内容在右 | 图文说明 |

!!! example "实战：选择合适的版式"
    1. 开始 → 新建幻灯片 → 选择版式
    2. 或右键左侧缩略图 → 版式 → 选择
    
    不要所有幻灯片都用"标题和内容"——用"节标题"分隔章节，用"两栏内容"做对比，用"空白"做创意排版。

## 1.3 配色方案

### 色彩心理学速查

| 颜色 | 情感联想 | 适用场景 |
|:---|:---|:---|
| **蓝色** | 专业、信任、稳定 | 企业报告、金融、科技 |
| **红色** | 激情、紧急、警告 | 强调重点、警示信息 |
| **绿色** | 成长、环保、安全 | 环保主题、正面指标 |
| **橙色** | 活力、创新、友好 | 创意提案、团队介绍 |
| **深灰/黑** | 权威、正式、高端 | 正式场合、高端品牌 |
| **白色** | 简洁、纯净、空间 | 留白、现代简约风格 |

### 配色方案设计

| 配色类型 | 说明 | 示例 |
|:---|:---|:---|
| **单色** | 同一色相不同明度 | 深蓝 + 中蓝 + 浅蓝 |
| **类比色** | 色轮上相邻的颜色 | 蓝 + 青 + 绿 |
| **互补色** | 色轮上相对的颜色 | 蓝 + 橙（用于强调） |
| **三色** | 色轮上等距的三种颜色 | 红 + 黄 + 蓝 |

!!! tip "60-30-10 法则"
    主色占 60%（背景、大面积区域），辅助色占 30%（标题、图表），强调色占 10%（重点标注、CTA 按钮）。

## 1.4 字体选择

### 字体搭配原则

| 用途 | 推荐字体（中文） | 推荐字体（英文） |
|:---|:---|:---|
| 标题 | 微软雅黑、思源黑体 | Arial、Calibri、Segoe UI |
| 正文 | 微软雅黑、宋体 | Calibri、Georgia |
| 代码/数据 | 等线、Consolas | Consolas、Courier New |

!!! warning "字体使用禁忌"
    - 一份演示文稿中字体不超过 **2~3 种**
    - 避免使用装饰性过强的字体（如 Comic Sans）
    - 确保字体在投影仪上清晰可读（无衬线字体优先）
    - 如果需要在其他电脑上演示， **嵌入字体** （文件 → 选项 → 保存 → 嵌入字体）

## 1.5 快速排版工具

### 对齐工具

| 工具 | 快捷键/路径 | 用途 |
|:---|:---|:---|
| 参考线 | `Alt+F9` | 显示/隐藏可拖拽的参考线 |
| 网格线 | `Shift+F9` | 显示/隐藏网格 |
| 智能参考线 | 自动 | 拖拽时显示红色虚线对齐提示 |
| 对齐工具 | 格式 → 对齐 | 左对齐、居中、横向分布等 |
| 选择窗格 | `Alt+F10` | 管理重叠对象的层级和可见性 |

!!! example "实战：快速均匀排列 5 个图标"
    1. 选中 5 个图标
    2. 格式 → 对齐 → 顶端对齐
    3. 格式 → 对齐 → 横向分布
    
    5 个图标瞬间等距排列，比手动拖拽精确 10 倍。

## 要点总结

- [ ] 理解 CRAP 设计原则，能识别和修正常见设计问题
- [ ] 掌握 6×6 法则，避免幻灯片变成 Word 文档
- [ ] 能根据演示场景选择合适的版式
- [ ] 了解配色和字体的基本原则
- [ ] 熟练使用对齐工具和参考线进行精确排版
