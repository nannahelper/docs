# 第 4 章：动画与过渡

> **从"一次性展示"到"节奏化叙事"——用动画控制信息流**

动画不是为了让 PPT"炫酷"，而是为了控制信息出现的节奏。好的动画让观众在正确的时刻看到正确的内容，而不是一次性被信息淹没。

---

## 4.1 动画的四种类型

| 类型 | 图标颜色 | 作用 | 典型用途 |
|:---|:---|:---|:---|
| **进入** | 绿色 | 元素出现在幻灯片上 | 逐条显示要点 |
| **强调** | 黄色 | 已显示的元素发生变化 | 突出关键数据 |
| **退出** | 红色 | 元素从幻灯片上消失 | 切换展示内容 |
| **动作路径** | 蓝色 | 元素沿指定路径移动 | 流程演示 |

!!! tip "动画选择原则"
    - 优先使用 **淡入/淡出** 和 **擦除** ——它们最不分散注意力
    - 避免使用 **弹跳**、**旋转**、**脉冲** 等夸张动画（除非有特殊目的）
    - 同一演示中动画风格保持一致

---

## 4.2 动画时间线控制

### 三种启动方式

| 启动方式 | 说明 | 适用场景 |
|:---|:---|:---|
| **单击时** | 单击鼠标才触发 | 演讲者控制节奏 |
| **与上一动画同时** | 与上一个动画一起播放 | 多个元素同时出现 |
| **上一动画之后** | 上一个动画结束后自动播放 | 自动播放序列 |

### 动画窗格

动画 → 动画窗格（`Alt+A+C`）打开时间线视图。

!!! example "实战：制作逐条显示的要点列表"
    1. 选中文本框
    2. 动画 → 添加动画 → 进入 → 淡入
    3. 效果选项 → 按段落
    4. 动画窗格中展开所有动画
    5. 设置第一个为"单击时"，其余为"上一动画之后"
    6. 设置每个动画的延迟为 0.3 秒

    效果：单击一次，所有要点依次淡入，节奏流畅。

### 持续时间与延迟

| 参数 | 建议值 | 说明 |
|:---|:---|:---|
| 持续时间 | 0.25~0.5 秒 | 太快看不清，太慢让人不耐烦 |
| 延迟 | 0.1~0.3 秒 | 连续动画之间的间隔 |
| 文字动画 | 按词/按字母 | 适合标题，不适合正文 |

## 4.3 高级动画技巧

### 触发器

让动画由特定对象的点击触发，而非按顺序播放。

!!! example "实战：点击按钮显示详细信息"
    1. 创建"查看详情"按钮（形状 + 文字）
    2. 选中详细信息文本框 → 添加进入动画（淡入）
    3. 动画 → 触发 → 通过单击 → 选择"查看详情"按钮

    演示时点击按钮，详细信息才出现——实现交互式演示。

### 动画刷

类似格式刷，将一个对象的动画复制到另一个对象。

!!! example "实战：使用动画刷"
    1. 选中已设置好动画的对象
    2. 动画 → 动画刷
    3. 点击目标对象

    双击动画刷可以连续应用到多个对象。

### 组合动画

一个对象可以添加多个动画，实现复杂效果。

!!! example "实战：图片飞入并放大强调"
    1. 选中图片
    2. 添加动画 → 进入 → 飞入（自左侧）
    3. 添加动画 → 强调 → 放大/缩小
    4. 动画窗格中，第二个动画设为"上一动画之后"

## 4.4 幻灯片过渡

### 过渡 vs 动画

| 对比维度 | 动画 | 过渡 |
|:---|:---|:---|
| 作用对象 | 幻灯片内的元素 | 幻灯片之间 |
| 设置位置 | 动画选项卡 | 切换选项卡 |
| 触发方式 | 单击/自动/触发 | 切换到该幻灯片时 |

### 过渡类型选择

| 过渡类型 | 风格 | 适用场景 |
|:---|:---|:---|
| **淡入/淡出** | 简洁优雅 | 通用、正式场合 |
| **推入** | 方向感 | 表示内容递进 |
| **覆盖** | 方向感 | 表示内容切换 |
| **揭开** | 柔和过渡 | 章节过渡 |
| **平滑** | 高级过渡 | 需要 Morph 效果时 |

!!! warning "过渡使用禁忌"
    - 不要每页用不同的过渡效果——显得杂乱
    - 避免使用"涡流"、"碎片"等夸张过渡
    - 正式商务演示建议只用"淡入/淡出"或"推入"

### Morph 过渡

**Morph**（平滑）是 PowerPoint 最强大的过渡效果，能自动在两个幻灯片之间创建平滑动画。

!!! example "实战：使用 Morph 制作缩放定位效果"
    1. 第一张幻灯片：放一张完整的产品图片
    2. 复制幻灯片（`Ctrl+D`）
    3. 第二张幻灯片：放大图片的某个细节部分
    4. 切换 → 平滑

    演示时从第一张切换到第二张，图片自动平滑放大到细节区域。

!!! tip "Morph 的更多用法"
    - 文字位置变化 → 文字飞入效果
    - 形状大小变化 → 缩放动画
    - 颜色变化 → 渐变效果
    - 旋转角度变化 → 旋转动画
