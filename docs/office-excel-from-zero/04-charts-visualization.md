# 第 4 章：图表与可视化

> **从"数字表格"到"数据故事"——用图表让数据自己说话**

一图胜千言。但选错图表类型，比没有图表更糟糕。本章教你如何选择合适的图表，并制作专业级的数据可视化。

---

## 4.1 图表类型选择指南

### 按数据关系选择图表

| 数据关系 | 推荐图表 | 示例场景 |
|:---|:---|:---|
| **比较** | 柱形图、条形图 | 各产品销售额对比 |
| **趋势** | 折线图、面积图 | 月度销售趋势 |
| **构成** | 饼图、堆积柱形图、树状图 | 市场份额分布 |
| **分布** | 散点图、直方图、箱线图 | 客户年龄分布 |
| **关系** | 散点图、气泡图 | 广告投入 vs 销售额 |
| **地理** | 地图图表 | 各省销售热力图 |

!!! warning "饼图的使用限制"
    饼图只适合展示 **5~7 个类别** 的占比。类别太多时使用条形图替代。 **永远不要使用 3D 饼图** ——3D 效果会扭曲面积比例，造成视觉误导。

### 图表元素清单

一份专业的图表应包含以下元素：

| 元素 | 必要性 | 说明 |
|:---|:---:|:---|
| 图表标题 | 必须 | 简洁描述图表内容 |
| 坐标轴标题 | 推荐 | 标注单位（如"万元"、"人"） |
| 数据标签 | 视情况 | 数据点少时直接标注数值 |
| 图例 | 多系列时必须 | 放在图表下方或右侧 |
| 网格线 | 可选 | 建议使用浅灰色，不要太显眼 |
| 数据表 | 视情况 | 需要同时展示精确数值时使用 |

## 4.2 图表高级定制

### 组合图：双轴图表

当两个数据系列的量级差异很大时（如销售额 1000 万 vs 增长率 20%），使用双轴图表。

!!! example "实战：销售额与增长率组合图"
    1. 选中数据 → 插入 → 组合图 → 创建自定义组合图
    2. 销售额：簇状柱形图，主轴
    3. 增长率：带数据标记的折线图，次轴
    4. 调整次轴刻度使折线在图表中部
    5. 主轴标题：销售额（万元）
    6. 次轴标题：增长率（%）

### 图表配色原则

| 原则 | 说明 |
|:---|:---|
| **少即是多** | 颜色不超过 5 种 |
| **语义一致** | 同一指标在不同图表中使用相同颜色 |
| **对比突出** | 重点数据用醒目颜色，其余用灰色 |
| **色盲友好** | 避免红绿搭配，使用蓝橙替代 |
| **品牌一致** | 使用公司品牌色系 |

!!! tip "自定义图表模板"
    设置好图表格式后，右键图表 → 另存为模板（.crtx）。此后插入图表时可以在"模板"中找到。

## 4.3 迷你图：单元格内趋势

### 三种迷你图

| 类型 | 用途 | 示例 |
|:---|:---|:---|
| **折线迷你图** | 展示趋势 | 12 个月销售走势 |
| **柱形迷你图** | 比较数值 | 各月销售额对比 |
| **盈亏迷你图** | 展示正负 | 月度盈亏状态 |

!!! example "实战：在数据表旁添加趋势迷你图"
    1. 选中要放置迷你图的单元格
    2. 插入 → 迷你图 → 折线图
    3. 数据范围：选择一行 12 个月的数据
    4. 确定
    5. 拖拽填充柄向下复制到其他行

    迷你图可以像普通单元格一样复制和填充。

## 4.4 动态图表

### 使用名称管理器创建动态数据源

当数据会持续增加时，使用 OFFSET 和 COUNTA 创建动态命名区域，图表自动包含新增数据。

!!! example "实战：自动扩展的图表数据源"
    1. 公式 → 名称管理器 → 新建
    2. 名称：SalesData
    3. 引用位置：
    ```
    =OFFSET(Sheet1!$B$2,0,0,COUNTA(Sheet1!$B:$B)-1,1)
    ```
    4. 右键图表 → 选择数据 → 编辑系列 → 系列值改为 `=Sheet1!SalesData`

    此后在 B 列新增数据，图表自动更新。

### 使用下拉列表切换图表视图

结合数据验证和 CHOOSE/INDEX 函数，创建可交互的图表。

!!! example "实战：按产品切换图表"
    1. 创建下拉列表（产品 A、产品 B、产品 C）
    2. 使用 INDEX 或 CHOOSE 函数根据下拉选择返回对应数据
    3. 图表绑定到公式返回的动态数据区域
